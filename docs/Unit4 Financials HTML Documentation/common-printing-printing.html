PrintingService

Namespace: http://www.coda.com/common/schemas/printing/printing-13.0/webservice
WSDL: common-printing-printing.wsdl
Path: services/common/printing/printing-13.0

The following methods are available on this endpoint:

  • MultiPrint
  • Print
  • PrintPreview
  • TestPrint
  • WritePayCollectData

MultiPrint

Input Parameters

MultiPrintRequest
Name Minimum Maximum Type Description
PrintFormatMasterCode 1 1 typeGeneralCode The code for the print format master.
Test 1 1 boolean Specifies whether or not this print preview request uses test data: x for alphanumeric characters, 9 for numeric characters and dd/mm/yy for dates.
OutputDeviceMasterCode 0 1 typeGeneralCode The code for the output device master.
PrintContent 1 unbounded PrintContent The data to print.
LogType 0 1 typeGenericOon The type of log to produce.
LogTitle 1 1 string The title for the log.
Product 0 1 string The Coda application requesting the print.
DocumentSplit 0 1 boolean Specifies whether multiple documents for a single customer should be printed seperately.

Return Value

MultiPrintResponse
Name Minimum Maximum Type Description
SuccessList 1 1 SuccessList A list of the printing request outcomes.
LogKey 0 1 LogKey The log key that identifies the log for this print run.

PrintRequest
Name Minimum Maximum Type Description

This method returns no data.
