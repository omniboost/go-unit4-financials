UserPreferencesService

Namespace: http://www.coda.com/common/schemas/userpreferences/userpreferences-12.0/webservice
WSDL: common-userpreferences-userpreferences.wsdl
Path: services/common/userpreferences/userpreferences-12.0

The following methods are available on this endpoint:

  • AddMasterFavourite
  • GetMasterFavourites
  • UpdateMasterFavourites
  • UpdateMasterFavouritesFromList

AddMasterFavourite

Input Parameters

AddMasterFavouriteRequest
Name Minimum Maximum Type Description
MasterFavourite 1 1 MasterFavourites This contains the Favourites list you want to update.
MasterFavouriteElement 1 1 MasterFavouritesListElement This contains the items in a Favourites list you want to update.

Return Value

AddMasterFavouriteResponse
Name Minimum Maximum Type Description
MasterFavourites 1 1 MasterFavourites The Favourites list that you want to add.

GetMasterFavourites

Input Parameters

GetMasterFavouritesRequest
Name Minimum Maximum Type Description
Key 1 1 FavouritesKey Contains the key identifying the Favourites list you want to retrieve.

Return Value

GetMasterFavouritesResponse
Name Minimum Maximum Type Description
MasterFavourites 1 1 MasterFavourites Contains the Favourites list you have retrieved from the database.

UpdateMasterFavourites

Input Parameters

UpdateMasterFavouritesRequest
Name Minimum Maximum Type Description
MasterFavourites 1 1 MasterFavourites Contains the information for the Favourites list you want to update.

Return Value

This method returns no data.

UpdateMasterFavouritesFromList

Input Parameters

UpdateMasterFavouritesFromListRequest
Name Minimum Maximum Type Description
MasterFavourites 1 1 MasterFavourites The original Favourites list.
MasterFavouritesList 1 1 MasterFavouritesListData The Favourites list you want to merge.

Return Value

UpdateMasterFavouritesFromListResponse
Name Minimum Maximum Type Description
MasterFavourites 1 1 MasterFavourites The Favourites list that you attempted to update.
