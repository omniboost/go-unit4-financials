SelectorService

Namespace: http://www.coda.com/efinance/schemas/selector/selector-13.0/webservice
WSDL: finance-selector-selector.wsdl
Path: services/finance/selector/selector-13.0

The following methods are available on this endpoint:

  • Select
  • SelectDetails

Select

Input Parameters

SelectRequest
Name Minimum Maximum Type Description
SelType 1 1 SelectorType Specifies the source of the data from which the selection is made. The data source is determined by the combination of data class and data location.
GslCode 1 1 typeGeneralCodeB The code of the selector master you want to use. GslCode is used to look up hints, and also overrides the SelType above.
RowLimit 0 1 typeULong The maximum number of rows you want your search to return
ApplicationFilter 1 1 VocListData The search criteria to be applied by the application when selecting the data. These are as set up in the selector master.
UserFilter 1 1 VocListData The additional search criteria specified by the user, to be applied when selecting the data. This may be, for example, a document code or a range of codes.
Columns 1 1 CellList The columns representing vocabulary items that you want the server to return.

Return Value

SelectResponse
Name Minimum Maximum Type Description
Selection 0 1 Selection The selected data returned to you for browsing.

SelectDetails

Input Parameters

SelectDetailsRequest
Name Minimum Maximum Type Description
SelType 1 1 DetailsLocation Indicates the source of the data, which may be retrieved from Books, Intray, Archive or Self.
GslCode 1 1 typeGeneralCodeB The code of the selector master you want to use. GslCode is used to look up hints, and also overrides the SelType above.
WholeDocuments 1 1 boolean If TRUE, specifies that you want to browse whole documents for the lines selected.
RowLimit 0 1 typeULong The maximum number of rows you want your search to return.
ApplicationFilter 1 1 VocListData The search criteria to be applied by the application when selecting the data. These are as set up in the selector master.
UserFilter 1 1 VocListData The additional search criteria specified by the user, to be applied when selecting the data. This may be, for example, a document code or a range of codes.
HierarchyFilter 0 1 VocListData The additional search criteria which will be applied to select data within specified hierarchies.
FlexiFieldFilter 1 1 FlexiFieldVocListData The additional search criteria which will be applied to select data from specified flexi-field masters.
Columns 1 1 HeaderColumnList The columns representing vocabulary items that you want the server to return.

Return Value

SelectDetailsResponse
Name Minimum Maximum Type Description
SelectionWithFlexiFields 0 1 SelectionWithFlexiFields The selected data, including flexi-fields, returned to you for browsing.
