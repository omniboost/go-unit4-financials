DiaryService

Namespace: http://www.coda.com/efinance/schemas/diary/diary-13.0/webservice
WSDL: finance-diary-diary.wsdl
Path: services/finance/diary/diary-13.0

The following methods are available on this endpoint:

  • Add
  • AddComment
  • Browse
  • GetComments
  • GetDefaultSelector
  • GetDiary
  • GetInfo
  • List
  • Update

Add

Input Parameters

AddRequest
Name Minimum Maximum Type Description
Entry 1 1 Entry This element contains details about the diary event you want to add.
Phasing 0 1 Phasing This element contains details about the rules applied when the diary event is replicated, and set to recur at regular intervals.
Comment 0 1 Comment This element contains the comments for the diary event you want to add.

Return Value

AddResponse
Name Minimum Maximum Type Description
Keys 1 1 AddKeyList The keys identifying the diary events added to the database by the 'Adddry' request.

AddComment

Input Parameters

AddCommentRequest
Name Minimum Maximum Type Description

Comment 1 1 Comment Contains the comment text that you want to add to the diary event.

Return Value

AddCommentResponse
Name Minimum Maximum Type Description
Key 1 1 Key The key identifying the diary event to which the comment was added.
CommentData 0 1 CommentData Contains details about when the diary event comment was created, and by which user.

BrowseResponse
