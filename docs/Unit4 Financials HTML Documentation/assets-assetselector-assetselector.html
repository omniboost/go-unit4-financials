SelectAssets

Input Parameters

SelectAssetsRequest
Name Minimum Maximum Type Description
SelType 1 1 DetailsLocation The code for the selector master used to select assets from the database.
IncludeChildren 1 1 boolean Specifies whether children of the selected parent assets should be included in the selction.
RowLimit 0 1 typeULong The maximum number of rows to be returned. The default is 1000.
ApplicationFilter 1 1 VocListData A list of criteria for selecting assets supplied by the system.
UserFilter 1 1 VocListData A list of criteria for selecting assets supplied by the user using a generic selector master.
Columns 1 1 CellList Each column specifies an item of data that is returned for each selected asset.

Return Value

SelectAssetsResponse
Name Minimum Maximum Type Description
Selection 1 1 Selection Contains the assets that are selected from the database.
