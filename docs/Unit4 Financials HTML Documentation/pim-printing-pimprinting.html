PIMPrintingService

Namespace: http://www.coda.com/pim/schemas/printing/pimprinting-12.0/webservice
WSDL: pim-printing-pimprinting.wsdl
Path: services/pim/printing/pimprinting-12.0

The following methods are available on this endpoint:

  • BackgroundPrint
  • Print
  • PrintPreview

BackgroundPrint

Input Parameters

BackgroundPrintRequest
Name Minimum Maximum Type Description
DocumentOutputMaster 1 1 DocumentOutputMaster The document output master to use.
DocumentKeyList 1 1 DocumentKeyList Contains key information identifying the documents to output.
LogTitle 1 1 typePimLogtitle The title of the log file in which details of the output process will be stored.
LogDetailLevel 1 1 typePimLogDetailLevel The level of detail that will be recorded in the log.
DocumentSelector 1 1 Selector The selector to use if no document keys are supplied.

Return Value

This method returns no data.

Print

Input Parameters

PrintRequest
Name Minimum Maximum Type Description
DocumentKeyList 1 1 DocumentKeyList Contains key information identifying the documents to output.
OutputDeviceCode 1 1 typeGeneralCode The code of the document output master to use.
PrintFormatCode 1 1 typeGeneralCode The code of the print format to use.

Return Value

This method returns no data.

PrintPreview

Input Parameters

PrintPreviewRequest
Name Minimum Maximum Type Description
DocumentKey 1 1 DocumentKey The key for the document to preview.
PrintFormatCode 1 1 typeGeneralCode The code for the print format master to use.

Return Value

PrintPreviewResponse
Name Minimum Maximum Type Description
PdfFile 1 1 PdfFile Contains a PDF file of the document you asked to preview.
