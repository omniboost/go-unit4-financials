Path: services/wsadapter/authenticate/authenticate-1.0

The authentication service.

The following methods are available on this endpoint:

  • Authenticate

Authenticate

Authenticate with Coda

Input Parameters

AuthenticateRequest
Name Minimum Maximum Type Description
NewPassword 0 1 string By specifying a new password the request will update the Coda user master with the new password.
SuppressDefaultUser 0 1 boolean If a domain user has a default Coda user mapped, authentication would normally use this if no Coda user code were specified. Pass true here in order to bypass this defaulting.

Return Value

AuthenticateResponse
Name Minimum Maximum Type Description
User 0 1 string The code of the Coda user authenticated
UserTimestamp 0 1 long The timestamp of the authenticated Coda user.
Company 0 1 string The code of the company logged into.
Capability 0 1 string The code of the capability master used by the Coda user.
CapabilityTimestamp 0 1 int The timestamp of the capability master.
Companies 0 unbounded Company If no company was specified in the request and no default company is declared on the Coda user master, the response will contain a list of possible company codes.
Users 0 unbounded User If no Coda user code was specified and the authenticating user has permission to authenticate as a number of Coda user masters, and there was either no default user or SuppressDefaultUser was true , the response will contain a list of possible user codes.
PasswordExpiryDate 0 1 date The date the Coda user master's password will expire.
PasswordExpiryWarning 0 1 boolean Whether the client should warn the user their password is soon to expire.
RecordLogoff 0 1 boolean Whether to record the log off.
FunctionalSecurityHash 0 1 string A hash of the functional security.
Session 0 1 string The session ID
Reserved 0 1 boolean Reserved for Coda use.

Complex Types

Company

Information about a company. Used to allow a client to provide an authenticating user with a friendly list of companies to choose from.

Child elements

Name Minimum Maximum Type Description

ShortName 0 1 string The short name of the company.

Child elements

Name Minimum Maximum Type Description
