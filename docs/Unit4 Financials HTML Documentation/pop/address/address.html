DefaultDeliveryAddressRequest

The DefaultDeliveryAddressRequest type has Request as its base type.

DefaultDeliveryAddress. Finds the default delivery address. This will either come from the company, or from the given department. If no address can be found (or the department cannot be found) then this function will return 'null'. The Department Element Level parameter is optional. If not given, then the service may still succeed, if an element with the given code exists at exactly one level.

Child elements

Name Minimum Maximum Type Description
CmpCode 1 1 typeCode The code of the company in which the address exists.
DeptElmCode 1 1 typeCode The code of the department element to which the address relates.
DeptElmLevel 0 1 typeElmLevelB The level of the department element to which the address relates.
