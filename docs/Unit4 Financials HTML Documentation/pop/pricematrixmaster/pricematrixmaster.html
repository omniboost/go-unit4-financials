UpdateRequest

The UpdateRequest type has Request as its base type.

Update. This service will ensure that existing versions of the master are not modified. If the incoming master contains more than one version, then only the highest version will be used. This will be compared to the latest version on the existing master, and if there is any difference, then a new version will be added to the master.

GetLogRequest

The GetLogRequest type has Request as its base type.

GetLog. Returns a master. Certain parts of the master can be used to form the log on the client. For some reason it doesn't seem possible to return PriceMatrixLogItems which contains just what is required, ie list of log items detailing a subset of the price matrix item, one for each version in the master. Should it become possible to get a good build while using PriceMatrixLogItems as the return type, then this can be changed.
