http://www.coda.com/pop/schemas/popselector

Root Elements

POPSelectorMasterRequest

This service allows you to maintain selector masters.

POPSelectorMasterResponse

This is the response that is returned when you maintain selector masters.

POPSelectorRequest

This service allows you to retrieve data according to selection criteria that you specify

POPSelectorResponse

This service holds the response to your request to retrieve data according to the selection criteria that you specified.

Complex Types

PopSelectionCount

This element holds information about the data returned by a selector.

Child elements

Name Minimum Maximum Type Description
SelType 1 1 SelectorType Indicates the type of selector: the data class you are using, and the source of the data, retrieved from Books, Intray, Archive or Self.
Header 1 1 SelectionRow Contains vocabulary items (representing columns) for the display of data, which is determined by the display criteria in the presenter master which was specified. Only those vocabulary items included in the presenter are used in the header.
Count 1 1 int The number of rows returned.
RowLimitExceeded 1 1 boolean If TRUE, specifies that the data returned is subject to a limit, since the response to your selection criteria exceeded the allowed number of rows.

VocabKeys

The fields used to filter the data that was returned.

Child elements

Name Minimum Maximum Type Description
KeyVocabs 1 1 CellList The vocabulary items used in the filter list.
KeyFields 1 1 CellList The values used in the filter list.

PopSelection

The selection criteria.

Child elements

Name Minimum Maximum Type Description
Selection 1 1 Selection The selection criteria.
KeyFields 1 1 VocabKeys The fields used to filter the data that was returned.
