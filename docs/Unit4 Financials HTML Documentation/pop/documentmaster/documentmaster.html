This property is used to hold all of the information associated with the general tab part of the Document Master.

This property is used to hold all of the information associated with the requisitioning tab part of the Document Master.

The sequence rule can either be automatic or uncontrolled. When automatic there must be an open number range, from which the number is allocated. When uncontrolled the user can manually enter a number. This number is then checked for uniqueness when saving or posting a document. Once the master is used (i.e. documents are created using this master), the sequence rule cannot be changed anymore.
