http://www.coda.com/pop/schemas/printing

Root Elements

POPPrintingRequest

This service allows you to specify options for printing and issuing Procurement documents.

POPPrintingResponse

This service holds the specified options for printing and issuing Procurement documents.

PrintingOptions

Child elements

Name Minimum Maximum Type Description
LogTitle 0 1 typeLogText The log title for this printing or issuing session.
LogType 0 1 typeCtOneOfN The log type, that is, Printing.

Reissuing 1 1 boolean Specifies whether this is a reissuing process.
DocumentFormatKeyList 1 1 DocumentFormatKeyList The list of the keys that identify the document and format details for a printing or issuing session.
ToXML 1 1 boolean Specifies that the output goes to an XML file for printing or issuing by fax or B2B.
XmlLocation 0 1 string The location of the XML file.

DocumentsPerFile 0 1 short The maximum number of documents to combine into a single repository file.
OverridePrintFormat 0 1 typeCodeB The code of a print format master to override the print format master specified elsewhere.
