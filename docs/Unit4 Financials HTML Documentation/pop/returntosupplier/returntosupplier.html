DiscSurchType 0 1 typePopInputDiscountsurcharge

Whether the discSurchValue refers to a discount or a surcharge.

DiscSurchValue 0 1 Money

The discount/surcharge value on the line in document/line currency. Will be negative in the case of a discount and positive in the case of a surcharge. May be calculated from settings on the item master or overidden by the user (either by setting a value or a percentage).

DiscSurchValueHome 0 1 Money

The discount/surcharge value on the line in home currency. Calculated from discSurchValue using the appropriate exchange rates.

DiscSurchValueDual 0 1 Money

The discount/surcharge value on the line in dual currency. Calculated from discSurchValue using the appropriate exchange rates. Only present if company is configured with a dual currency.

NetOfTaxValue 0 1 Money

Net of tax value of the line in document/line currency. Calculated as netValue + discSurchValue.

NetOfTaxValueHome 0 1 Money

Net of tax value of the line in home currency. Calculated as netValueHome + discSurchValueHome.

NetOfTaxValueDual 0 1 Money

Net of tax value of the line in dual currency. Calculated as netValueDual + discSurchValueDual. Only present if company is configured with a dual currency.

TaxValue 0 1 Money

Tax value of the line in document/line currency. Calculated from netOfTaxValue using the rate from the tax master.

TaxValueHome 0 1 Money

Tax value of the line in home currency. Calculated from taxValue using the appropriate exchange rates.

TaxValueDual 0 1 Money

Tax value of the line in dual currency. Calculated from taxValue using the appropriate exchange rates. Only present if company is configured with a dual currency.

IrrecTaxValue 0 1 Money

Irrecoverable tax value of the line in document/line currency. Calculated from taxValue using the irrecoverable rate from the tax master.

IrrecTaxValueHome 0 1 Money

Irrecoverable tax value of the line in home currency. Calculated from irrecTaxValue using the appropriate exchange rates.

IrrecTaxValueDual 0 1 Money

Irrecoverable tax value of the line in dual currency. Calculated from irrecTaxValue using the appropriate exchange rates. Only present if company is configured with a dual currency.

GrnAccountAllocationId 1 1 typeDbGuid

PercentageReceived 0 1 typePercentageb

The percentage received against this account. (Calculated)

QuantityReturnedToDate 0 1 typeNumberb

Quantity returned to date

PriceReturnedToDate 0 1 Money

Price returned to date. For value-only items, this is the amount returned. Otherwise, this is the unit price.

QuantityAvailableToReturn 0 1 typeNumberb

The quantity of items for this account that may be returned. (Calculated)

PriceAvailableToReturn 0 1 Money

The price of items for this account that may be returned. (Calculated) For value-only items, this is the amount that may be returned. Otherwise, this is the unit price.

InvoicedQuantity 0 1 typeNumberb

InvoicedValue 0 1 Money

PostedQuantity 0 1 typeNumberb

PostedValue 0 1 Money
