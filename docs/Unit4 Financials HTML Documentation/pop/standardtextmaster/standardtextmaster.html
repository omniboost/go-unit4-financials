AddUpdateRequest

The AddUpdateRequest type has Request as its base type.

AddUpdate. Replaces the usual add and update services. If a new text is sent then the version number will be automatically incremented. This ensures that versions are sequential, starting at 1, and that previous versions cannot be overwritten.
