PopAppParams

The PopAppParams type has CommonAppParams as its base type.

This element holds the information relating to application parameters.

Child elements

Name Minimum Maximum Type Description
EIMExtension 0 1 typeCodeB The code of the procurement system master in Invoice Matching that provides the link between the procurement and invoice matching systems. This is mandatory.
RfqSuppWebServer 0 1 typeTextB The location of the supplier portal site. This must be completed if users will be issuing RFQs to suppliers.
RfqSuppEmailTransport 0 1 typeCodeB The message transport master used to send e-mails to suppliers. This must be completed if users will be issuing RFQs to suppliers.
RfqSuppEmailSubject 0 1 typeTextB The subject line used for e-mail messages sent to suppliers. This must be completed if users will be issuing RFQs to suppliers.
RfqSuppEmailMessage 0 1 typeTextB The text used in the body of e-mail messages sent to suppliers. This must be completed if users will be issuing RFQs to suppliers.
RfqSenderName 0 1 typeTextB The sender's name to be used for RFQs, if the RFQ sender e-mail below is used.
RfqSenderEMail 0 1 typeTextB The e-mail address to be used as the sender's address for RFQs, if none is set on the user master of the user who is issuing the RFQ.
OneGRNPerOrder 0 1 typeBoolean Specifies that each order selected for receiving will generate a separate GRN.
