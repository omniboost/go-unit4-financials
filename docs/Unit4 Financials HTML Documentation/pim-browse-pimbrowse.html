EmailSnapshot

Input Parameters

EmailSnapshotRequest
Name Minimum Maximum Type Description
TransportMasterCode 1 1 typeGeneralCode The code of the message transport master that will be used to e-mail the snapshot.
Recipients 1 1 RecipientList The list of Coda users who will receive the snapshot.
Subject 1 1 typeTextfieldb The subject that will appear on the e-mail.
Priority 1 1 typePimBrsEmailPriority The priority that the e-mail will take.
Content 1 1 typeMultitextb The message that will appear in the e-mail.
SnapshotFilename 1 1 string The filename of the snapshot's .html file.
Snapshot 1 1 string The snapshot data.

Return Value

EmailSnapshotResponse
Name Minimum Maximum Type Description
SuccessfulRecipients 1 1 RecipientList This element contains a list of Coda users to whom the snapshot was e-mailed successfully.
FailedRecipients 1 1 RecipientList This element contains a list of Coda users to whom the snapshot was e-mailed unsuccessfully.
