POPPrintingService

Namespace: http://www.coda.com/pop/schemas/printing/popprinting-13.0/webservice
WSDL: pop-printing-popprinting.wsdl
Path: services/pop/printing/popprinting-13.0

The following methods are available on this endpoint:

  • BackgroundPrint
  • GetDefaultPrintFormat
  • GetOutputDeviceList
  • Print
  • PrintPreview
  • PrintPreviewList
  • SelectBatch

BackgroundPrint

Input Parameters

BackgroundPrintRequest
Name Minimum Maximum Type Description
ProcessMasterCode 1 1 typeCode The Printing or Issuing process master.
LogTitle 1 1 typeLogText The log title for this printing or issuing session.
LogType 1 1 typeCtOneOfN The log type, that is, Printing.
Reissuing 1 1 boolean Specifies whether this is a reissuing process.
SelectorMaster 0 1 SelectorMaster The code of the selector master to use.
PrintFormatCode 0 1 typeCodeB The code of the print format master to use.

Return Value

This method returns no data.

GetDefaultPrintFormat

Input Parameters

GetDefaultPrintFormatRequest
Name Minimum Maximum Type Description
DocumentFormatKey 1 1 DocumentFormatKey The key that identifies the document and format details for this session.

Return Value

GetDefaultPrintFormatResponse
Name Minimum Maximum Type Description
DefaultPrintFormat 1 1 typeCode The default print format master retrieved from the database.

GetOutputDeviceList

Input Parameters

GetOutputDeviceListRequest
Name Minimum Maximum Type Description
DocumentFormatKeyList 1 1 DocumentFormatKeyList A list of keys that identify the documents for which you want a list of output devices. The format information in the key is not used and the list of output devices in the key is ignored.

Return Value

GetOutputDeviceListResponse
Name Minimum Maximum Type Description
OutputDeviceList 1 1 OutputDeviceMasterItemList A list of the output devices specified by the documents in the list of document keys in the request.
