PostByKeys

Input Parameters

PostByKeysRequest
Name Minimum Maximum Type Description
LogTitle 1 1 typePimLogtitle A title for the log file that stores details of the posting process.
LogDetailLevel 1 1 typePimLogDetailLevel The level of detail to be recorded in the log file.
InvoiceKeyList 1 1 DocumentKeyList Contains key information identifying the documents to be posted.
PostToCurrentYearPeriod 1 1 boolean Determines whether each document will be posted to the current year/period rather than the year/period specified on the document itself.

Return Value

This method returns no data.
