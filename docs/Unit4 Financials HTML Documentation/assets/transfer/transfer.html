PostingValues

Contains the values that will be posted (or have been posted) to the Books to record this transfer.

Child elements

Name Minimum Maximum Type Description
CapitalCostSource 1 1 typeMoney The amount credited to the capitalisation account of the source asset.
CapitalCostDestination 1 1 typeMoney The amount debited from the capitalisation account of the destination asset.
AccumulatedDepreciationSource 1 1 typeMoney The amount debited from the accumulated depreciation account of the source asset.
AccumulatedDepreciationDestination 1 1 typeMoney The amount credited to the accumulated depreciation account of the destination asset.
RevalCostSource 1 1 typeMoney The amount credited to the revalued capital cost account of the source asset. This posting is only made if the asset has been revalued and ComplexDisp is set to TRUE for the PostRuleMaster used by this asset.
RevalCostDestination 1 1 typeMoney The amount debited from the revalued capital cost account of the destination asset. This will use the same account code as RevalCostSource except that the cost centre element will be the new cost centre. This posting is only made if the asset has been revalued and ComplexDisp is set to TRUE for the PostRuleMaster used by this asset.
RevalDepnSource 1 1 typeMoney The amount credited to the revalued accumulated depreciation account of the source asset. This posting is only made if the asset has been revalued and ComplexDisp is set to TRUE for the PostRuleMaster used by this asset.
RevalDepnDestination 1 1 typeMoney The amount debited from the revalued accumulated depreciation account of the destination asset. This will use the same account code as RevalDepnSource except that the cost centre element will be the new cost centre. This posting is only made if the asset has been revalued and ComplexDisp is set to TRUE for the PostRuleMaster used by this asset.
RevalResSource 1 1 typeMoney The amount credited to the revaluation reserve account of the source asset. This posting is only made if the asset has been revalued and ComplexDisp is set to TRUE for the PostRuleMaster used by this asset.
RevalResDestination 1 1 typeMoney The amount debited from the revaluation reserve account of the destination asset. This will use the same account code as RevalResSource except that the cost centre element will be the new cost centre. This posting is only made if the asset has been revalued and ComplexDisp is set to TRUE for the PostRuleMaster used by this asset.
YTDDepreciationSource 1 1 typeMoney This posting is only made when performing a cost centre transfer and TransferDepreciation is set to TRUE for the transfer process. It is the amount credited to the depreciation charge account of the source asset (the old cost centre).
YTDDepreciationDestination 1 1 typeMoney This posting is only made when performing a cost centre transfer and TransferDepreciation is set to TRUE for the transfer process. It is the amount debited from the depreciation charge account of the destination asset (the new cost centre).
CategoryOut 1 1 typeMoney This posting is only made when performing a category transfer. It is the amount credited to reverse out the original WIP capitalisation postings.
CategoryIn 1 1 typeMoney This posting is only made when performing a category transfer. It is the amount debited for the new assets created by the category transfer process.
AcceleratedDepreciation 1 1 typeBool If TRUE, indicates that the Accelerated Depreciation flag is set on the category master.
AcceleratedDepreciationProfit 1 1 typeBool If TRUE, indicates that the Accelerated Depreciation value is the profit value.
NegateAcceleratedDepreciation 1 1 typeBool Not currently used.
TransferResidualAmount 1 1 typeMoney The amount of residual value transferred to the destination asset.
