TransactionDetail

Contains the details of a single transaction. Each transaction applies to a single asset and is a record of an event which resulted in changing the asset's cost or Net Book Value.

Child elements

Name Minimum Maximum Type Description
CmpCode 1 1 typeGeneralCode The company code of the asset.
Code 1 1 typeAstCode The asset code.
UserCode 1 1 typeGeneralCode The name of the user who ran the original process, for example, the name of the user who started the depreciation run.
BookCode 1 1 typeGeneralCode The code for the book master used to post the asset.
LinkedAsset 0 1 typeAstCode The code of an asset that is linked as a parent or child to this asset.
DeprnCode 0 1 typeGeneralCode If the transaction is a Depreciation, the code of the depreciation master is contained here.
DocumentCode 0 1 typeGeneralCode The code for the document master used to post the transaction. If a posting in Financials was made, this element (in conjunction with DocumentNumber, DocLineNumber and CmpCode) identifies the line.
DocumentNumber 0 1 typeDocNum The number of the document. If a posting in Financials was made, this element (in conjunction with DocumentCode, DocLineNumber and CmpCode) identifies the line.
DocLineNumber 0 1 int The number of the document line. If a posting in Financials was made, this element (in conjunction with DocumentCode, DocumentNumber and CmpCode) identifies the line.
TransactionType 1 1 typeTran The transaction type, for example Depreciate or Revalue.
EventType 1 1 typeEvent The name of the event, for example Depreciation or Revaluation.
TransactionDate 1 1 dateTime The date when the transaction took place.
TransactionYear 1 1 short The year of the financial transaction.
TransactionPeriod 1 1 short The period of the financial transaction.
Value 1 1 typeNum The value of the transaction.
RunNumber 0 1 int Contains the run number for the current transaction. All the transactions posted in a single depreciation run will have the same run number.
TransactionNumber 0 1 int Contains the transaction number for the current transaction. This is used to uniquely identify a transaction.
RealTransactionYear 0 1 short The year of the asset transaction.
RealTransactionPeriod 0 1 short The period of the asset transaction.
CostValue 1 1 typeNum The cost value of the transaction. An increase to the asset cost is shown as a positive value, while a reduction is shown as a negative value.
DepreciationValue 1 1 typeNum The depreciation value of the transaction. Typically these will be shown as negative values because depreciation usually reduces the NBV.
MemoValue 1 1 typeNum The value of a transaction that does not affect the NBV.
CreditAccount 1 1 typeAcCodeB The credit account for any associated Financials posting.
DebitAccount 1 1 typeAcCodeB The debit account for any associated Financials posting.
CancelEvent 1 1 boolean Indicates whether the transaction is a cancelling transaction.
CancelRef 1 1 int The run number of the cancelling process.
PreviousCostCentre 1 1 typeAcCodeB The previous cost centre if this transaction is a cost centre transfer.
Quantity 1 1 typeULong The quantity specified if this transaction is a part disposal or transfer.

Child elements

Name Minimum Maximum Type Description

TransactionCompleteDetail

The TransactionCompleteDetail type has TransactionDetail as its base type.

Child elements

Name Minimum Maximum Type Description
LastKnownYear 1 1 short The last known year that a depreciation run was performed on this asset.
IsFinalPeriod 1 1 boolean Denotes whether this transaction is in the final period of the assets life.
DoesNotReduceNbv 1 1 typeBool Denotes that this transaction will not reduce the asset's Net Book Value.
AlternativeAccount 0 1 typeAcCodeB An alternative account code that this transaction will be posted to. This is an alternative to the account code specified on the posting rule master.
AlternativeDocument 0 1 typeGeneralCodeB An alternative document that will be used to post this transaction. This is an alternative to the document specified on the posting rule master.
OriginalRunNumber 0 1 int The original depreciation run number of a transaction that is to be cancelled using the Cancel depreciation function.
OriginalTransactionNumber 0 1 int The original depreciation transaction number of a transaction that is to be cancelled using the Cancel depreciation function.
LogID 0 1 int The log number.
