NumRuleMaster

This element holds a number rule master.

Child elements

Name Minimum Maximum Type Description
TimeStamp 1 1 short The TimeStamp value for this number rule master in the database.
CmpCode 1 1 typeGeneralCode The code for the company in which the number rule master is being maintained.
Code 1 1 typeGeneralCode The code for this number rule master.
Name 1 1 typeName The full name for this number rule master.
ShortName 1 1 typeShortNameB The abbreviated name for this number rule master.
Prefix 1 1 string The string to use as the prefix for asset numbers generated by this number rule master. The prefix can contain characters, digits and symbols excluding the * and ? wildcards
NumberLength 1 1 short The number of digits in the numerical part of the asset number.
StartNumber 1 1 int The first number that can be generated by this number rule master
NextNumber 1 1 int The next number that will be generated by this number rule master. This value is read-only.
EndNumber 1 1 int The last number that can be generated by this number rule master.
Ammendable 1 1 boolean Specifies whether users can amend asset numbers that are generated by this number rule master. This is FALSE by default, and cannot be set if Temporary is set to TRUE.
Temporary 1 1 boolean Set to TRUE to make this number rule master generate temporary numbers. This value can be TRUE for only one number rule master in each Coda company.
