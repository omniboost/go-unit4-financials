CancelDepreciation

This element contains parameters specifying which depreciation should be cancelled.

Child elements

Name Minimum Maximum Type Description
SelectorCode 1 1 typeGeneralCode The code of the selector master that you want to use to select assets for cancelling depreciation. Only assets selectors can be specified.
SelectorMaster 1 1 SelectorMaster This element holds the selector master specified by SelectorCode.
CancelToYearPeriod 1 1 string The period up to which depreciation is cancelled. All depreciation transactions in the selected period and later will be cancelled. For example, if you have run depreciation for periods 1-6, and you specify period 4 here, depreciation will be cancelled for periods 4-6 inclusive.
DocYearPeriod 1 1 string The year and period to which the cancel depreciation transactions are posted.
DocumentDate 1 1 dateTime The date of the document which posts the 'reversing' financial transaction to the Financials Books.
LogTitle 1 1 string The title of the log file storing details of the cancel depreciation process. This is mandatory.
AggregateAccumDepn 1 1 boolean Indicates whether 'reversing' financial transactions posted to the same accumulated depreciation account are summarised.
AggregateDepnCharge 1 1 boolean Indicates whether 'reversing' financial transactions posted to the same depreciation charge account are summarised.
CmpCode 1 1 typeGeneralCode The code for the company in which depreciation is being cancelled.
RunNumber 1 1 int A number identifying this cancel depreciation process.
UseSpecifiedPeriod 1 1 boolean If set, uses the specified year and period for posting the cancelling transaction.
