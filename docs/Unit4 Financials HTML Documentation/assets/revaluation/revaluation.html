Revaluation

This element contains the parameters which can be specified when revaluing assets.

Child elements

Name Minimum Maximum Type Description
Selector 1 1 typeText The selector master that you want to use to select assets for revaluation.
SelectorMaster 0 1 SelectorMaster This element holds the selector master specified by Selector.
RevaluationType 1 1 typeRevaluationDetails Specify whether you want to revalue the selected assets by a value, a percentage, or using a custom calculation.
Value 1 1 typeMoney The amount by which the selected asset(s) should be revalued. This will be a negative amount if the asset's value has decreased.
Percent 1 1 typePercent The percentage by which the selected asset(s) should be revalued. This will be a negative amount if the asset's value has decreased.
Custom 1 1 typeText The custom revaluation rule to use for revaluing the selected asset(s).
PostingYearPeriod 1 1 typeBaseYearPeriod The year and period to which the revaluation postings will be made.
PostingDocumentDate 1 1 typeDate The document date that will be used for the revaluation postings.
RevalueChildren 1 1 typeBool Set this to TRUE to revalue dependent child assets along with their parents. Set this to FALSE to revalue the parent assets but not their dependent children.
CapitalCostRevaluation 1 1 typeBool Set this to TRUE to aggregate document lines with the same account code for capital cost revaluation into a single line. Document lines with the same account code but using different documents cannot be aggregated.
AccumulatedDepreciationRevaluation 1 1 typeBool Set this to TRUE to aggregate document lines with the same account code for accumulated depreciation revaluation into a single line. Document lines with the same account code but using different documents cannot be aggregated.
RevaluationReserve 1 1 typeBool Set this to TRUE to aggregate document lines with the same account code for the revaluation reserve into a single line. Document lines with the same account code but using different documents cannot be aggregated.
PostingDetails 1 1 RevaluationPostings Contains the values that will be posted by the revaluation process.
IsReviewed 1 1 typeBool Specifies whether the revaluation values are reviewed before being posted.
PostingRunID 1 1 RunIdList This element contains a list of companies in which the revaluation process will be run.
LogTitle 0 1 string The log file in which details of this revaluation process will be stored. This field is mandatory.
RevaluationDate 1 1 typeDate The date of the revaluation.
