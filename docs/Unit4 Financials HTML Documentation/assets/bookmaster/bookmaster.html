BookMaster

This element holds a book master.

Child elements

Name Minimum Maximum Type Description
CmpCode 1 1 typeGeneralCode The code for the company in which the book master is being maintained.
Code 1 1 typeGeneralCode The code for this book master.
LongName 1 1 typeName The full name for this book master.
ShortName 1 1 typeShortName The abbreviated name for this book master.
TimeStamp 1 1 short The TimeStamp value for this book master in the database.
PostingBook 1 1 typeBool Defines this book master as the posting book for the company.
Posted 1 1 typeBool Specifies whether or not, this book has been used to post assets.
BookType 0 1 typeBook Indicates whether this book master is the primary (posting) book, a secondary book or a currency book.
AssociatedBookCode 0 1 typeGeneralCode When BookType is currency, this is the code of the primary book or the secondary book with which this currency book is associated.
CurCode 0 1 typeGeneralCode When BookType is currency, this identifies the currency in which values in this currency book will be held.

Child elements

Name Minimum Maximum Type Description
MaxKeys 1 1 int The maximum number of items to be returned.
ShortName 0 1 string The short name of the book master that you want to select. This can be wildcarded to return multiple book masters.
Key 1 1 Key Contains key information for the specified book masters.
