Root Elements

AssetTransactionDetail

This element contains the details of the transactions that will be posted to the Assets database tables.

FinancialTransactionDetail

This element contains details of the transactions that will be posted to the Financials database tables.

ReviewTransactions

Contains the transaction information generated for the asset.

Complex Types

ReviewTransactions

Contains the transaction information generated for the asset.

Child elements

Name Minimum Maximum Type Description
FinancialTransactions 1 1 FinancialTransactionList Lists the financial transactions.
AssetTransactions 1 1 AssetTransactionList Lists the asset transactions.

FinancialTransactionList

Lists the financial transactions.

Child elements

Name Minimum Maximum Type Description
List 1 unbounded FinancialTransactionDetail Contains a list of financial transactions generated.

FinancialTransactionDetail

Contains a list of financial transactions generated.

Child elements

Name Minimum Maximum Type Description
DocCode 1 1 typeCode The code for the document master used to post the asset.
DocNumber 1 1 typeText The number of the document.
AccountCode 1 1 typeAcCode The code for the account to which the asset is posted.
DocValue 1 1 typeMoney The value of the document.
LineSense 1 1 typeText The line sense: either debit or credit.
CurrencyCode 0 1 typeGeneralCodeB The code of the currency for the posting.
Period 0 1 typePeriodB The period in which the posting is made.
Currencies 0 1 Currencies Holds the element currency information for the document line.(Information is only held for currencies that will be posted, that is are set up as currency books on the primary book.)
DualCur 1 1 typeCode The code of the dual currency for the posting.
DualValue 1 1 typeMoney The document value in dual currency.
BookCode 1 1 typeCode The Assets book code.

AssetTransactionList

Lists the asset transactions.

Child elements

Name Minimum Maximum Type Description
List 1 unbounded AssetTransactionDetail Contains a list of asset transactions generated.

AssetTransactionDetail

Contains a list of asset transactions generated.

Child elements

Name Minimum Maximum Type Description
Type 1 1 typeTran The type of the last transaction.
Asset 1 1 typeCode The code for the asset.
YearPeriod 1 1 typeBaseYearPeriod The year and period in which the asset will be posted.
Book 1 1 typeCode The asset book affected by the posting.
Document 1 1 typeCode Reference to the document that will be posted.
Rule 1 1 typeCode The code for the depreciation rule used.
Value 1 1 typeMoney The value of the transaction.
CostValue 1 1 typeMoney The cost value of the transaction. An increase to the asset cost is shown as a positive value, while a reduction is shown as a negative value.
DepreciationValue 1 1 typeMoney The depreciation value of the transaction. Typically these will be shown as negative values because depreciation usually reduces the NBV.
MemoValue 1 1 typeMoney The value of a transaction that does not affect the NBV.
