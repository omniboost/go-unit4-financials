Root Elements

Disposal

This element contains the parameters which can be specified when disposing assets.

DisposalPostings

Contains the postings that will be generated (or have been generated) by the current disposal request.

DisposalRequest

This service allows you to dispose assets.

DisposalResponse

This is the response that is returned when you dispose assets.
