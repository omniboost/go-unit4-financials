PostRuleMaster

This element holds a posting rule master.

Child elements

Name Minimum Maximum Type Description
TimeStamp 1 1 short The TimeStamp value for this posting rule master in the database.
CmpCode 1 1 typeAcCode The code for the company in which the posting rule master is being maintained.
Code 1 1 typeAcCode The code for this posting rule master.
Name 1 1 typeName The full name for this posting rule master.
ShortName 1 1 typeShortNameB The abbreviated name for this posting rule master.
DocCapnCode 1 1 typeGeneralCode The code of the document master to use for recording postings generated when capitalising assets.
DocDispCode 1 1 typeGeneralCode The code of the document master to use for recording postings generated when disposing assets.
DocDepnCode 1 1 typeGeneralCode The code of the document master to use for recording postings generated when depreciating assets.
DocRevalCode 1 1 typeGeneralCode The code of the document master to use for recording postings generated when revaluing assets.
DocTranCode 1 1 typeGeneralCode The code of the document master to use for recording postings generated when transferring assets.
AcctCapital 1 1 typeAcCode The account to which capital values will be posted. This account is mandatory.
AcctRevenue 1 1 typeAcCodeB The account to which associated expenses will be posted. For example, if the asset is a car, you might need to record its delivery cost as an expense.
AcctDepnAccum 1 1 typeAcCode The balance sheet account to which accumulated depreciation will be posted. This account is mandatory.
AcctDepnCharge 1 1 typeAcCode The profit and loss account to which the depreciation cost will be posted. This account is mandatory.
CapCostReval 1 1 typeAcCodeB The account to which any revaluation of an asset's capital cost is posted.
DepnAccReval 1 1 typeAcCodeB The account to which any revaluation of an asset's accumulated depreciation is posted.
RevalRes 1 1 typeAcCodeB The account to which any difference between the revaluation of an asset's capital cost and the revaluation of an asset's accumulated depreciation is posted.
CapCostRevSold 1 1 typeAcCodeB The account to which the revalued capital cost of the asset would be posted if the asset was sold.
DepnAccRevSold 1 1 typeAcCodeB The account to which the revalued accumulated depreciation of the asset would be posted if the asset was sold.
RevalResSold 1 1 typeAcCodeB The account to which the revaluation reserve for the asset would be transferred if the asset was sold.
DispProcAcct 1 1 typeAcCodeB The account to which proceeds resulting from an asset's disposal, will be posted.
DispProfAcct 1 1 typeAcCodeB The account to which any profits resulting from an asset's disposal, will be posted.
DispLossAcct 1 1 typeAcCodeB The account to which any losses resulting from an asset's disposal, will be posted.
CapCostSold 1 1 typeAcCodeB The account to which an asset's capital cost will be posted upon disposal.
AccumDepSold 1 1 typeAcCodeB The account to which an asset's accumulated depreciation will be posted upon disposal.
ProceedsTxAcct 1 1 typeAcCodeB The account to which the proceeds value (with the opposite sign) is posted to balance the document. This account is used only when ComplexDisp is set to TRUE.
PLTxAcct 1 1 typeAcCodeB The account to which the profit or loss value (with the opposite sign) is posted to balance the document. This account is used only when ComplexDisp is set to TRUE.
AddnAcctDepnCharge 1 1 typeAcCodeB The profit and loss account to which any depreciation cost resulting from a special depreciation rule is posted.
ComplexDisp 1 1 typeBool Set to TRUE if you want postings to be made to the transfer accounts on disposal of assets. When this is FALSE, disposal of assets does not generate postings to the transfer accounts. If you have upgraded from the Coda Windows Asset product, set this to TRUE to have disposal postings behave as they did in that product.
AnnuityDocCode 0 1 typeGeneralCodeB The code of the document master to use when posting annuity (interest) values.
AnnuityInterest 0 1 typeAcCodeB The account code to which the annuity (interest) values will be posted.
SuspenseSetAtRuntime 0 1 boolean Indicates whether or not the suspense account is specified at runtime. If this is set to FALSE, you must supply the suspense account code in the SuspenseAcct element.
SuspenseAcct 0 1 typeAcCodeB The suspense account to use at runtime.
ExpenseSetAtRuntime 0 1 boolean Indicates whether or not the expense account is specified at runtime. If this is set to FALSE, you must supply the expense account code in the AcctRevenue element.
UseAlternateAccounting 0 1 boolean Indicates whether standard accounting or alternative accounting is used for simple disposals.
AlternateDispCapCostUseOriginal 0 1 boolean When selling an asset, this indicates whether the original capitalisation account is used for the disposed cost. If this is FALSE, you must specify an alternative account in the AlternateDispCapCostAcct element.
AlternateDispCapCostAcct 0 1 typeAcCodeB The alternative account to use for disposed cost values when selling an asset.
AlternateDispAccumDepnUseOriginal 0 1 boolean When selling an asset, this indicates whether the original depreciation account is used for the disposed accumulated depreciation. If this is FALSE, you must specify an alternative account in the AlternateDispAccumDepnAcct element.
AlternateDispAccumDepnAcct 0 1 typeAcCodeB The alternative account to use for disposed accumulated depreciation values when selling an asset.
SuppressProfitLossPosting 0 1 boolean For complex disposals, indicates whether postings to the profit and loss accounts are suppressed when selling as asset.
UseScrapAccounts 0 1 boolean Indicates whether separate disposal accounts are used for scrap disposals. When FALSE, the same disposal accounts are used for both sale and scrap.
AlternateDispCapCostScrapUseOriginal 0 1 boolean When scrapping an asset, this indicates whether the original capitalisation account is used for the disposed cost. If this is FALSE, you must specify an alternative account in the AlternateDispCapCostScrapAcct element.
AlternateDispCapCostScrapAcct 0 1 typeAcCodeB The alternative account to use for disposed cost values when scrapping an asset.
AlternateDispAccumDepnScrapUseOriginal 0 1 boolean When scrapping an asset, this indicates whether the original depreciation account is used for the disposed accumulated depreciation. If this is FALSE, you must specify an alternative account in the AlternateDispAccumDepnScrapAcct element.
AlternateDispAccumDepnScrapAcct 0 1 typeAcCodeB The alternative account to use for disposed accumulated depreciation values when scrapping an asset.
CapCostScrapped 0 1 typeAcCodeB For complex disposals, this is the account to which the assets' capital cost would be posted if the assets were scrapped.
AccumDepScrapped 0 1 typeAcCodeB For complex disposals, this is the account to which the assets' accumulated depreciation would be posted if the assets were scrapped.
SuppressScrapProfitLossPosting 0 1 boolean For complex disposals, this indicates whether postings to profit, loss, and profit/loss transfer accounts are suppressed when scrapping assets.
DispProfScrapAcct 0 1 typeAcCodeB The account to which any profits resulting from the scrapping of assets, will be posted.
DispLossScrapAcct 0 1 typeAcCodeB The account to which any losses resulting from the scrapping of assets, will be posted.
PLTxScrapAcct 0 1 typeAcCodeB For complex disposals, this is the account to which the profit or loss value (with the opposite sign) is posted to balance the document when assets are being scrapped.
DocReclassCode 1 1 typeGeneralCode The code of the document master to use for recording postings generated when transferring WIP assets to another category for capitalisation (category transfers).
AcctOutOfWIP 1 1 typeAcCodeB The account to which reversing values will be credited to reverse out the original WIP capitalisation postings.
AcctInToCap 1 1 typeAcCodeB The account from which capital values for the new assets created by the category transfer process will be debited.
AcceleratedDepnProfit 0 1 typeAcCodeB The account to which accelerated depreciation profit values will be posted.
AcceleratedDepnLoss 0 1 typeAcCodeB The account to which accelerated depreciation loss values will be posted.
