This service allows you to cancel transaction events.

This is the response that is returned when you cancel events.

Complex Types

CancelEvents

This element contains parameters specifying which events should be cancelled.

Child elements

Name Minimum Maximum Type Description
SelectorCode 1 1 typeGeneralCode The code of the selector master that you want to use to select assets for event cancellation. Only assets selectors can be specified.
SelectorMaster 0 1 SelectorMaster This element holds the selector master specified by SelectorCode.
AssetCodes 0 1 AssetKeyList This element lists the assets that meet the selection criteria and will be subject to event cancellation.
CancelToYearPeriod 1 1 string The period up to which events will be cancelled. All events in the selected period and later will be cancelled. For example, if the current year and period is 2008/6 and you specify 2008/4 in this field, then any events that took place in 2008 periods 4-6 inclusive (excluding capitalisation) will be cancelled.
UseSpecifiedPeriod 1 1 boolean When TRUE, the transaction will be posted to the year/period specified in the DocYearPeriod element. When FALSE, it will be posted to the year/period of the transaction itself.
DocYearPeriod 1 1 string The year and period to which the cancelling transactions are posted.
DocumentDate 1 1 dateTime The date of the document which posts the cancelling financial transactions to the Financials Books.
LogTitle 1 1 string The title of the log file storing details of the cancellation process. This is mandatory.
CmpCode 1 1 typeGeneralCode The code for the company in which events are being cancelled.
RunNumber 1 1 int A number identifying this cancellation process.
