Complex Types

CancelRevaluation

This element contains parameters specifying which revaluation should be cancelled.

Child elements

Name Minimum Maximum Type Description
SelectorCode 1 1 typeGeneralCode The code of the selector master that you want to use to select assets for cancelling revaluation. Only assets selectors can be specified.
SelectorMaster 1 1 SelectorMaster This element holds the selector master specified by SelectorCode.
CancelToYearPeriod 1 1 string The period up to which revaluation is cancelled. All revaluation events in the selected period and later will be cancelled. For example, if you revalued your assets in period 6, and you specify period 4 here, all revaluations in periods 4-6 inclusive will be cancelled unless prevented by the presence of other transactions.
DocYearPeriod 1 1 string The year and period to which the cancel revaluation transactions are posted.
DocumentDate 1 1 dateTime The date of the document which posts the cancelling financial transactions to the Financials Books.
LogTitle 1 1 string The title of the log file storing details of the cancel revaluation process. This is mandatory.
AggrCapCostRevalued 1 1 boolean Indicates whether financial transactions that would be posted to the same capital cost revaluation account are summarised.
AggrAccumDepnRevalued 1 1 boolean Indicates whether financial transactions that would be posted to the same accumulated depreciation revaluation account are summarised.
AggrRevalReserve 1 1 boolean Indicates whether financial transactions that would be posted to the same revaluation reserve account are summarised.
CmpCode 1 1 typeGeneralCode The code for the company in which revaluation is being cancelled.
RunNumber 1 1 int A number identifying this cancel revaluation process.
