Depreciation

This element contains the parameters which can be specified when running depreciation.

Child elements

Name Minimum Maximum Type Description
SelectorCode 1 1 typeGeneralCode The code of the selector master that you want to use to select assets for depreciation. Only assets selectors can be specified.
SelectorMaster 1 1 SelectorMaster This element holds the selector master specified by SelectorCode.
YearPeriod 1 1 string The year and period up to which depreciation is calculated.
DocumentDate 1 1 dateTime The date of the document which posts the depreciation.
PostZeroValues 1 1 boolean Set this to TRUE to post zero values of depreciation.
AggregateDepreciation 1 1 boolean Set this to TRUE to summarise postings that are made to the same Accumulated Depreciation account.
AggregateCharge 1 1 boolean Set this to TRUE to summarise postings that are made to the same Depreciation Charge account.
PostToLastPeriod 1 1 boolean Set this to TRUE to post asset transactions to the year/period specified by YearPeriod. When FALSE, asset transactions are posted to the year/period to which the depreciation applies.
LogTitle 1 1 string A title for the log file that stores details of the depreciation run. This value is mandatory.
DetailedLog 1 1 boolean Set this to TRUE to have details of the depreciation calculations appended to the log file.
ErrorsOnly 1 1 boolean Specifies whether only error messages are saved to the log file. When TRUE, warning messages, success messages, and information messages are not logged.
SuppressValidation 1 1 boolean Set this to TRUE to suppress additional validation that can be performed before the depreciation is posted; standard validation is still performed. Setting this to TRUE speeds up the depreciation process.
ContinueAfterError 1 1 boolean Set this to TRUE if you want the depreciation process to continue if an error is encountered. When this is FALSE, the depreciation process will stop if an error is encountered.
CmpCode 0 1 string The code of the company in which depreciation will be run.
RunNumber 0 1 int A number identifying the depreciation run.
LogId 0 1 int A number identifying the log in which the details of this depreciation process are stored.
