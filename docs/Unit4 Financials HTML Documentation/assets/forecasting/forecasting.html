Forecasting

This element contains the parameters which can be specified when forecasting depreciation.

Child elements

Name Minimum Maximum Type Description
SelectorCode 1 1 typeGeneralCode The code of the selector master that you want to use to select assets for forecasting.
SelectorMaster 1 1 SelectorMaster This element holds a selector master.
YearPeriodFrom 1 1 string The year and period at the start of the range for which you want to run forecasting.
YearPeriodTo 1 1 string The year and period at the end of the range for which you want to run forecasting.
AggregateDepreciation 1 1 boolean Set this to TRUE to summarise postings that are made to the same Accumulated Depreciation account.
AggregateCharge 1 1 boolean Set this to TRUE to summarise postings that are made to the same Depreciation Charge account.
CalculateYearly 1 1 boolean Set this to TRUE to calculate the forecast figures yearly rather than by period. Calculating the figures yearly will be quicker, but less detailed, than calculating the figures by period.
LogTitle 1 1 string A title for the log file that stores details of the forecasting run. This is mandatory.
DetailedLog 1 1 boolean Set this to TRUE to have details of the forecast depreciation calculations appended to the log file.
ErrorsOnly 1 1 boolean Specifies whether only error messages are saved to the log file. When TRUE, warning messages, success messages, and information messages are not logged.
Destination 1 1 string Specify whether the results are exported to a CSV file or to a Coda non-maintained balance code.
BalanceCode 1 1 string The non-maintained balance code to which you want the results to be exported. This needs to be completed only if the Destination is set to be a Coda non-maintained balance code.
IncludeAssetTransactions 1 1 boolean Specifies whether to include asset transactions in the forecast information. When FALSE, only financial transactions are included.
IncludeAdditionalCurrencies 1 1 boolean Specifies whether values in associated currency books are also forecast.
IncludeIntrayAssets 1 1 boolean When True, Intray assets will be included in the selection providing that the selector master specified in SelectorCode includes 'Intray' as a data location.
