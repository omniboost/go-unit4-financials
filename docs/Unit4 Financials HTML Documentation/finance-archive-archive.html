ArchiveService

Namespace: http://www.coda.com/efinance/schemas/archive/archive-2.0/webservice
WSDL: finance-archive-archive.wsdl
Path: services/finance/archive/archive-2.0

The following methods are available on this endpoint:

  • Abort
  • GetArchiveDocumentsInformation
  • GetProgress
  • Pause
  • Restart
  • Resume
  • StartArchivingDocuments

Abort

Input Parameters

AbortRequest
Name Minimum Maximum Type Description
Key 1 1 ProgressKey The key, or special reference, which identifies the process you want to abort. This is given to you when you start the process.

Return Value

AbortResponse
Name Minimum Maximum Type Description
Key 1 1 ProgressKey The key, or special reference which identifies the process you want to abort.

GetArchiveDocumentsInformation

Input Parameters

GetArchiveDocumentsInformationRequest
Name Minimum Maximum Type Description
ArchiveHeaderMasterKey 1 1 Key Contains the key identifying the archive header master you want to use for archiving documents.

Return Value

GetArchiveDocumentsInformationResponse
Name Minimum Maximum Type Description
CmpCode 1 1 typeGeneralCode The code of the company in which the Archive process is running.
ArchiveHeaderMasterCode 1 1 typeGeneralCode The code of the header master being used to archive documents.
ArchiveDocumentsInformation 0 1 ArchiveDocumentsInformation Contains the default parameters and current status for the Archive Documents process. The parameters can be edited if required.

GetProgress

Input Parameters

GetProgressRequest
Name Minimum Maximum Type Description
Key 1 1 ProgressKey The key, or special reference, which identifies the process you started. This is given to you when you start the process.

Return Value

GetProgressResponse
Name Minimum Maximum Type Description
Key 1 1 ProgressKey The key, or special reference, which identifies the process you are sending the request to. This is given to you when you start the process.
Progress 0 1 ProgressData Contains information about the status and progress of the process you are running.
ArchiveMonitor 0 1 ArchiveMonitor Contains details of the progress made by the Archive process.
Error 0 1 EmbeddedReason Contains information about an error that occurred.

Return Value

Restart

Input Parameters

RestartRequest
Name Minimum Maximum Type Description

Return Value

RestartResponse
Name Minimum Maximum Type Description

ArchiveHeaderMasterCode 1 1 typeGeneralCode The code of the archive header master you specified for the restarted Archive process.
ProgressKey 0 1 ProgressKey The key, or special reference, given to you for requesting information about the progress of the Archive process.

StartArchivingDocuments

Input Parameters

StartArchivingDocumentsRequest
Name Minimum Maximum Type Description
ArchiveDocumentsParams 1 1 ArchiveDocumentsParams Contains the parameters for starting the Archive Documents process.

Return Value

StartArchivingDocumentsResponse
Name Minimum Maximum Type Description
CmpCode 1 1 typeGeneralCode The code of the company in which you are attempting to archive documents.
ArchiveHeaderMasterCode 1 1 typeGeneralCode The code of the archive header master you are using to archive documents.
BalanceWarning 0 1 BalanceWarning A warning that not all documents for the year have been archived. Archive Documents should be run before archiving balances.
ProgressKey 0 1 ProgressKey The key, or special reference, given to you for requesting information about the progress of Archive Documents.
