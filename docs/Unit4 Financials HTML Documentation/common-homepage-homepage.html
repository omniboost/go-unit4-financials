HomePageService

Namespace: http://www.coda.com/common/schemas/homepage/homepage-3.0/webservice
WSDL: common-homepage-homepage.wsdl
Path: services/common/homepage/homepage-3.0

The following methods are available on this endpoint:

  • Get
  • GetContent
  • GetFrame
  • GetFramesetContent
  • GetPage
  • Update
  • UpdateContent
  • UpdateFrame

Get

Input Parameters

This method has no input parameters.

Return Value

GetResponse
Name Minimum Maximum Type Description
Page 1 1 Page Contains the definition of the Home Page that has been retrieved from the database.
Contents 1 1 Contents Contains the content available in frames on this Home Page.
ContentProviders 1 1 ContentProviderList Contains details of the content provider masters available to this Home Page.

GetContent

Input Parameters

GetContentRequest
Name Minimum Maximum Type Description
ContentProviderCode 1 1 typeGeneralCode The code of the content provider master to use.
Configuration 1 1 FrameConfiguration This element holds the settings which may be configured for the frame.
LastUpdate 0 1 dateTime The date and time when the content was last updated.

Return Value

GetContentResponse
Name Minimum Maximum Type Description
Content 0 1 Content Contains the content that has been retrieved in accordance with the parameters specified in the corresponding Request.
ContentProvider 0 1 ContentProviderListItem Contains details of the content provider master used to provide this content.

GetFrame

Input Parameters

GetFrameRequest
Name Minimum Maximum Type Description
FrameID 1 1 string The ID of the frame you want to retrieve. Frame IDs are unique per frameset.

Return Value

GetFrameResponse
Name Minimum Maximum Type Description
Frame 0 1 Frame Contains the frame that has been retrieved from the database.
Content 0 1 Content Contains the content available in this frame.
ContentProvider 0 1 ContentProviderListItem Contains details of the content provider master used in this frame.

GetFramesetContent

Input Parameters

GetFramesetContentRequest
Name Minimum Maximum Type Description
FramesetIndex 1 1 int The index identifying the frameset.
Page 1 1 Page The Home Page containing the specified frameset.
ContentProviders 1 1 ContentProviderList The content provider masters available to this Home Page.

Return Value

GetFramesetContentResponse
Name Minimum Maximum Type Description
Page 1 1 Page Contains the Home Page specified in the corresponding Request.
Contents 1 1 Contents Contains the content available in frames on this Home Page.
ContentProviders 1 1 ContentProviderList Contains details of the content provider masters available to this Home Page.

GetPage

Input Parameters

This method has no input parameters.

Return Value

GetPageResponse
Name Minimum Maximum Type Description
Page 1 1 Page Contains the definition of the Home Page that has been retrieved from the database.
ContentProviders 1 1 ContentProviderList Contains details of the content provider masters available to this Home Page.

Update

Input Parameters

UpdateRequest
Name Minimum Maximum Type Description
Page 1 1 Page Contains the Home Page to be updated. This is the Home Page of the current user.

Return Value

UpdateResponse
Name Minimum Maximum Type Description
Page 1 1 Page Contains the updated Home Page. This is the Home Page of the current user.
Contents 1 1 Contents Contains the content available in frames on this Home Page.
ContentProviders 1 1 ContentProviderList Contains details of the content provider masters available to this Home Page.

UpdateContent

Input Parameters

UpdateContentRequest
Name Minimum Maximum Type Description
ContentID 1 1 string The number identifying the frame and content configuration to be refreshed.
LastUpdate 1 1 dateTime The date and time when this content configuration was last updated.

Return Value

UpdateContentResponse
Name Minimum Maximum Type Description
Content 0 1 Content Contains the refreshed content.
ContentProvider 0 1 ContentProviderListItem Contains details of the content provider master used to provide this content.

UpdateFrame

Input Parameters

UpdateFrameRequest
Name Minimum Maximum Type Description
Frame 1 1 Frame Contains the frame to be updated.

Return Value

UpdateFrameResponse
Name Minimum Maximum Type Description
Frame 1 1 Frame Contains the updated frame.
Content 0 1 Content Contains the content in this frame.
ContentProvider 0 1 ContentProviderListItem Contains details of the content provider master used to provide the content in this frame.
