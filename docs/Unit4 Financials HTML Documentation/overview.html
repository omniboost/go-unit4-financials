Unit4 Financials Web Services 14.000.0011

This document is the Web Service API specification for Unit4 Financials 14.000.0011.

Web Services provides an industry-standard interface to the complete suite of XMLi services over an internet connection using a generated URL for each service. This allows you to easily develop interfaces to update or validate CODA data or to extract information from CODA.
