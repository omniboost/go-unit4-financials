The reason code to be applied to the discrepancy.

ReasonDescription 0 1 typeDescription

The description for the reason code.

Value 1 1 Money

The overall discrepancy value to split across the lines.

Account 1 1 typePimAccountCode

The account code to be applied to each discrepancy line.

AccountMask 1 1 typeText

The account mask useed to select the actual lines to be used.

Comment 0 1 typeTextB

An optional comment to be added to each discrepancy.

Return Value

Name Minimum Maximum Type Description

Warnings 0 unbounded DocumentToLineDiscrepancyWarning The warnings is an out argument, so the server will put it here.
Group 1 1 Group The group is an out argument, so the server will put it here.
