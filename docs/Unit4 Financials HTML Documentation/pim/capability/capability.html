The amount of variation of currency details value that is tolerated for the user. Note that this value is in the home currency of the company currently being used.

TaxToleranceCapability 1 1 typeCapValueOrPercent

Whether there is a value, percentage, or no limit.

TaxToleranceValue 0 1 Money

The amount of correction allowed on differences between the entered tax and the generated tax expressed as a value. Note that this value is in the home currency of the company currently being used.

TaxTolerancePcnt 0 1 typePercentageb

The amount of correction allowed on differences between the entered tax and the generated tax expressed as a percentage.

AllowAccountModificationInMatching 0 1 typeBoolean

Determines whether or not the user is permitted to edit the account code on a given set of assignment line during interactive matching.

AlwaysApplyUserTolerances 0 1 typeBoolean

Should we always apply the user's tolerances.

DiscrepancyCreationToleranceMaster 0 1 ToleranceMasterKey

The discrepancy creation tolerance master to use for this capability. Note that the capability isn't within a particular company but this custom mapper will only return the tolerance master for the company we are operating in. This means we don't need to mess about looking through a list. If there is a tolerance master for this company it will be set, if not it will be null.

DiscrepancyAuthorisationToleranceMaster 0 1 ToleranceMasterKey

The discrepancy authorisation tolerance master to use for this capability. Note that the capability isn't within a particular company but this custom mapper will only return the tolerance master for the company we are operating in. This means we don't need to mess about looking through a list. If there is a tolerance master for this company it will be set, if not it will be null.

AccountSecuritySet 1 1 typeBoolean

Has the user got account security, either read or write lists set?

SecurityLevel 1 1 typeWord

Capability security level

PeriodSecurityGroup 1 1 typeWord

Capability period security group

AllowWorkflowOverride 1 1 typeBoolean

Whether the user can override the workflow set on an input document master when submitting documents to document authorisation workflow.

ModifyAuthUser 1 1 typeBoolean

Whether the user can modify the authorising user set on an input document, which is original set from the input document master.

TimeStamp 1 1 short The TimeStamp number of this Capability.
