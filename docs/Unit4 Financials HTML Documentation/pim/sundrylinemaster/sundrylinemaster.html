Complex Types

Level

Defines a level in an AccountCodeMapping. Depending upon the type of the Level different rules will be applied when building an account code from the mapping

Child elements

Name Minimum Maximum Type Description
LevelNumber 1 1 typeElmLevel

The level's number. Must be in the range 1 - 8

Type 1 1 typePimAccCodeMappingOptions

There are 8 levels in an AccountCodeMapping, one for each element in a valid CODA account. Each Level has a type which will drive the way an account code is built up

ElementCode 0 1 typeElmCode

The Element code associated with this Level (if any). This applies if the mapping type is an element code.
