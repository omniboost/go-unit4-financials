http://www.coda.com/pim/schemas/deferredmatch

Root Elements

PIMDeferredMatchSelectorRequest

This service allows you to select documents that will be included in a deferred matching process.

PIMDeferredMatchSelectorResponse

This is the response that is returned when you select documents that will be included in a deferred matching process.
