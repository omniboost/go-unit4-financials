Root Elements

PIMSelfBillingRequest

This service allows you to run self-billing.

PIMSelfBillingResponse

This is the response that is returned when you run self-billing.

PIMSelfBillingSelectorRequest

This service allows you to select documents that will be included in a self-billing process.

PIMSelfBillingSelectorResponse

This is the response that is returned when you select documents that will be included in a self-billing process.
