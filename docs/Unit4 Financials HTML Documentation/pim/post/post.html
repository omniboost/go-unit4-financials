Root Elements

PIMPostRequest

This service allows you to post purchase invoice documents to Financials.

PIMPostResponse

This is the response that is returned when you post purchase invoices to Financials.

PIMPostSelectorRequest

This service allows you to select documents for posting.

PIMPostSelectorResponse

This is the response that is returned when you select documents for posting.
