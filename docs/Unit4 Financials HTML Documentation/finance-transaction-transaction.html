GetDocument

Input Parameters

GetDocumentRequest
Name Minimum Maximum Type Description

Return Value

GetDocumentResponse
Name Minimum Maximum Type Description
Key 1 1 txnKey This element contains a key uniquely identifying the document you have retrieved.

Transaction 0 1 Transaction This element holds the financial document you have retrieved.
