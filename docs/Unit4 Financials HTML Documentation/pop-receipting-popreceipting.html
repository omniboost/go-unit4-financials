GenerateReceiptFromPurchaseOrder

Input Parameters

GenerateReceiptFromPurchaseOrderRequest
Name Minimum Maximum Type Description
InputTemplate 1 1 InputTemplateMasterKey

The Input Template that is used to create the receipts. This is a mandatory field.

DocumentMasterCode 1 1 typeCode

The document code to use when creating the document. If the template can be applied to multiple documents then use this field to specify which document to use. If the template can be applied to a single document then this field does not have to be populated as the function will retrieve the document code.

OrderKeys 1 1 PurchaseOrderKeyList The purchase orders for which GRNs are to be created.

Return Value

GenerateReceiptFromPurchaseOrderResponse
Name Minimum Maximum Type Description
GRNList 1 1 GoodsReceivedNoteList The GRNList is an out argument, so the server will put it here.
