Start

Input Parameters

StartRequest
Name Minimum Maximum Type Description
Params 1 1 StartParams Contains the parameters needed to select the ledgers to update.

Return Value

StartResponse
Name Minimum Maximum Type Description
CmpCode 1 1 typeGeneralCode The code of the company for which ledger updates are currently running.
UpdateMode 1 1 typeCtUpdateMode Specifies the update method being used. The available values are Single update (ledger information is updated only once) or Continuous update (ledger information is updated repeatedly at intervals specified in the IdleInterval field).
IdleInterval 0 1 typeULong The interval in hours between each ledger update. This option is available only when the UpdateMode is Continuous.
AllCompanies 1 1 boolean Specifies whether you are updating all ledgers for all companies to which you have access.
LedgerSelection 0 1 typeGeneralCodeB The code for the ledger master to update. This field is blank if all ledger masters for the current company are being updated.
AllLedgers 1 1 boolean Specifies whether you are updating all ledgers for the current company.
