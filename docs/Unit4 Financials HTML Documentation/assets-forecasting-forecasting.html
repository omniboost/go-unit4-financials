ForecastingService

Namespace: http://www.coda.com/eassets/schemas/forecasting/forecasting-12.0/webservice
WSDL: assets-forecasting-forecasting.wsdl
Path: services/assets/forecasting/forecasting-12.0

The following methods are available on this endpoint:

  • BalanceSave
  • LogsWithSpreadsheetData
  • Progress
  • SpreadsheetData
  • Start

BalanceSave

Input Parameters

BalanceSaveRequest
Name Minimum Maximum Type Description
CmpCode 1 1 string The code of the company in which forecasting is being performed.
RunNumber 1 1 int A number identifying the forecast depreciation run.
BalanceCode 1 1 string The Coda non-maintained balance code to which the forecast information will be saved.

Return Value

BalanceSaveResponse
Name Minimum Maximum Type Description
WrittenCount 1 1 int Contains the balance of the specified Coda non-maintained balance code.

LogsWithSpreadsheetData

Input Parameters

LogsWithSpreadsheetDataRequest
Name Minimum Maximum Type Description
CmpCode 1 1 string The code of the company in which you want to search for forecasting logs that have spreadsheet data available for download.

Return Value

LogsWithSpreadsheetDataResponse
Name Minimum Maximum Type Description
Response 0 1 LogsWithSpreadsheetDataList

Progress

Input Parameters

ProgressRequest
Name Minimum Maximum Type Description
CmpCode 1 1 string The code of the company in which forecasting is being performed.
LogID 1 1 int The log number.

Return Value

ProgressResponse
Name Minimum Maximum Type Description
PercentCompleteMessage 1 1 string A message indicating the percent complete of the forecasting process

SpreadsheetData

Input Parameters

SpreadsheetDataRequest
Name Minimum Maximum Type Description
CmpCode 1 1 string The code of the company in which forecasting is being performed.
RunNumber 1 1 int A number identifying the forecast depreciation run.
IncludeAssetTransactions 1 1 boolean Specifies whether to include asset transactions in the forecast information. When FALSE, only financial transactions are included.

Return Value

SpreadsheetDataResponse
Name Minimum Maximum Type Description
FinancialData 1 1 TemporaryDocumentDetailList Contains the financial transactions forecast data that has been generated for export to a spreadsheet file.
AssetData 0 1 TransactionTempDetailList Contains the asset transactions forecast data that has been generated for export to a spreadsheet file.
