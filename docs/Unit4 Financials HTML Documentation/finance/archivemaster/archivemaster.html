http://www.coda.com/efinance/schemas/archivemaster

Root Elements

ArchiveHeaderMaster

This element holds an archive header master.

ArchiveHeaderMasterRequest

This service allows you to maintain archive header masters.

ArchiveHeaderMasterResponse

This is the response that is returned when you maintain archive header masters.

Complex Types

ArchiveHeaderMasterCommon

Child elements

Name Minimum Maximum Type Description
CmpCode 1 1 typeGeneralCode The code of the company where the header master exists.
Code 1 1 typeGeneralCode The code of the archive header master.

DatabaseConnection

The element contains the database settings for a Billing external archive.

Child elements

Name Minimum Maximum Type Description
Username 1 1 typeDBConnectionField The name of the database user, which must be entered for external archives.
DatabaseName 0 1 typeDBConnectionField The name of the database, which is mandatory for some databases.
Password 0 1 typeDBConnectionFieldB The database password, which is mandatory for some databases.
PasswordUnset 1 1 boolean If TRUE, specifies that the password for this database in not set.
TableOwner 1 1 typeDBConnectionField The name of the table owner, which is mandatory for all databases.

JournalDetails

This element contains settings for the archive journal.

Child elements

Name Minimum Maximum Type Description
NormalJournal 1 1 typeGeneralCode The journal code used for non-turnover documents.
TurnoverJournal 1 1 typeGeneralCode The journal code used for turnover documents.
Description 0 1 typeDescr A description that appears on all journal lines.
PauseRequestSize 0 1 typePauseRequestSize The number of documents selected before the system checks to see if a request to pause the Archive process has been issued. The default size is 5000, and 1000 is the smallest number allowable.
SuppressZeroLines 1 1 boolean If TRUE specifies that lines with a line value and quantity of zero will not be stored on the archive journal.

ProcurementDetails

If TRUE specifies that you want to run Archive for Billing.

Child elements

Name Minimum Maximum Type Description
DisposalOfDocuments 1 1 typeCtAhmArchType Specifies whether the documents to be archived are moved from the main Billing database into archive tables, or whether they are purged from the database.
ExternalDatabase 0 1 DatabaseConnection The element contains the database settings for a Billing external archive.

ArchiveHeaderMaster

The ArchiveHeaderMaster type has ArchiveHeaderMasterCommon as its base type.

This element holds an archive header master.

Child elements

Name Minimum Maximum Type Description
TimeStamp 0 1 short The TimeStamp value for this archive header master in the database.
Name 1 1 typeName The full name for the archive header master.
ShortName 0 1 typeShortName The abbreviated name for the archive header master.
ExternalArchive 1 1 boolean If TRUE specifies that an external archive process is required for Financials data.
JournalDetails 0 1 JournalDetails This element contains settings for the archive journal.
ClusterSize 1 1 typeClusterSize The number of documents processed into a single archive journal. This must be set to at least 10.
DefaultSelector 0 1 typeGeneralCode The selector master code used as the default when starting to archive documents. This field may be left blank to select all appropriate documents.
SingleYear 0 1 short Indicates whether more than one year can be added to the same archive database. To specify a single year, enter the appropriate year. To allow more than one year, leave the field blank.
DeleteComments 1 1 boolean If TRUE specifies that comments will be deleted from the Books without copying them to the archive.
DeleteDiscounts 1 1 boolean If TRUE specifies that discount information will be deleted from the Books without copying it to the archive.
DeleteDiaryEvents 1 1 boolean If TRUE specifies that diary events and comments will be deleted from the Books without copying them to the archive.
DeleteTax 1 1 boolean If TRUE specifies that tax information will be deleted from the Books without copying it to the archive. This affects any VAT contributions memos, 1099 tax memos and tax turnover information.
DeletePay 1 1 boolean If TRUE specifies that payment information will be deleted from the Books without copying it to the archive.
ProcurementDetails 0 1 ProcurementDetails If TRUE specifies that you want to run Archive for Billing.
CoreExternalDatabase 0 1 DatabaseConnection This element contains the database settings for a Financials external archive.

ArchiveHeaderMasterStatus

The ArchiveHeaderMasterStatus type has ArchiveHeaderMasterCommon as its base type.

This element holds information about the current status of the header, and the documents which have been archived using this archive header.

Child elements

Name Minimum Maximum Type Description
CurrentState 1 1 typeCtArchiveState The current state of the archive header master, whether it is available, or already active.
CurrentUser 1 1 typeGeneralCode The user currently using this header master for an Archive process.
FromYearsAndPeriods 0 1 typePeriod The earliest year and period for the documents in the archive.
ToYearsAndPeriods 0 1 typePeriod The latest year and period for the documents in the archive.
FromDocumentDates 0 1 typeDate The dates of the earliest archive documents.
ToDocumentDates 0 1 typeDate The dates of the latest archive documents.
FromInputDates 0 1 typeDate The earliest input dates in the archive
ToInputDates 0 1 typeDate The latest input dates in the archive.
FromPaymentDates 0 1 typeDate The earliest payment dates in the archive.
ToPaymentDates 0 1 typeDate The latest payment dates in the archive.
ArchiveCreated 0 1 typeDate The date and time when the first journal was created.
LastUpdated 0 1 typeDate The date and time when the latest journal was created.
FirstNormalJournal 1 1 typeGeneralCode The code of the first journal written.
FirstNormalJournalNumber 1 1 typeDocNum The number of the first journal written.
FirstTurnoverJournal 1 1 typeGeneralCode The code of the first turnover journal written.
FirstTurnoverJournalNumber 1 1 typeDocNum The number of the first turnover journal written.
NumberOfDocsArchived 1 1 typeLong The total number of documents archived.
Paused 1 1 boolean If true, indicates that the Archive process using this header is paused.
