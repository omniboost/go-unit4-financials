Root Elements

BalanceMaster

This element holds a balance master.

This service allows you to maintain balance masters.

This is the response that is returned when you maintain balance masters.

BalanceMaster

This element holds a balance master.

Child elements

Name Minimum Maximum Type Description
CmpCode 1 1 typeGeneralCode The code of the company in which the balance masters exists.
TimeStamp 1 1 short The TimeStamp value for this balance master in the database.
Code 1 1 typeGeneralCode The code of the balance master.
Name 1 1 typeName The full name for the balance master.
ShortName 1 1 typeShortNameB The abbreviated name for the balance master.
CODAMaintained 1 1 boolean If TRUE, specifies that this balance is maintained by Financials.
DecimalPlaces 1 1 typeDecPlace The number of decimal places to which the balance is held.
BalMaintainedByCommitments 1 1 boolean If TRUE, specifies that the balance is maintained by the Commitment Accounting functionality in Procurement.

ReqKeys

Contains key information for requesting a number of balance masters from the database.

Child elements

Name Minimum Maximum Type Description
MaxKeys 1 1 int The maximum number of items to be returned.
Key 1 1 Key A balance master key.
EnableMasterSecurityFilter 0 1 boolean When TRUE, only balance masters to which the current user has read/write access are returned. A user's access to balance masters is controlled by the master security lists on the user's capability master.
