Root Elements

PayFormatMaster

This element contains a pay format master.

PayFormatMasterRequest

This service allows you to maintain pay format masters.

PayFormatMasterResponse

This is the response that is returned when you maintain pay format masters.

Complex Types

PayFormatMaster

This element contains a pay format master.

Child elements

Name Minimum Maximum Type Description
TimeStamp 1 1 short The TimeStamp value for this pay format master in the database.
Code 1 1 typeGeneralCode The code for this pay format master.
Name 1 1 typeName The full name for this pay format master.
ShortName 1 1 typeShortNameB The abbreviated name for this pay format master.
Attachment 1 1 Attachment Contains the information identifying the pay format file, which is stored in the Repository.
IsSepa 1 1 boolean Specifies whether this pay format conforms to SEPA guidelines.
OutputFileName 1 1 typePayFormatFileNameStrB Specifies the name of the output file that is generated for payments that use this pay format master.
CreateDate 0 1 typeDateB The date the record was added.
ModifyDate 0 1 typeDateB The date the record was modified.
User 0 1 typeGeneralCodeB The code of the user who added or modified the record.
