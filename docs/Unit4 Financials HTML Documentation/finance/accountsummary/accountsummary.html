http://www.coda.com/efinance/schemas/accountsummary

Root Elements

AccountSummaryRequest

This service allows you to retrieve account summaries.

AccountSummaryResponse

This is the response that is returned when you retrieve account summaries.

Complex Types

Key

This element is a key consisting of a company code, element level, element code and account summary master which uniquely identifies an account summary. It is used to specify search criteria for the retrieval of one or more account summaries.

Child elements

Name Minimum Maximum Type Description
CmpCode 1 1 typeGeneralCode The code for the company in which you want to view the account summary information.
ElmLevel 1 1 typeElmLevel The level of the element (1 to 8) for which you want to view the account summary information.
ElmCode 1 1 typeElmCode The code of the element for which you want to view the account summary information.
AsmCode 1 1 typeGeneralCode The code of the account summary master used to display the account summary information.

Summary

This element contains the summary data for each of the retrieved summaries.

Child elements

Name Minimum Maximum Type Description
SequenceNumber 1 1 int The number assigned to each control in the account summary.
Value 1 1 typeMoney The numeric value contained within the control.
Status 1 1 typeCtCtlValuStatus The status of the control.

SummaryList

A list of balance summary and details summary data values retrieved from the database.

Child elements

Name Minimum Maximum Type Description
Summary 0 unbounded Summary This element contains the summary data for each of the retrieved summaries.

AccountSummaryContext

This element contains the data that will be displayed in the account summary.

Child elements

Name Minimum Maximum Type Description
Element 0 1 Element The element for which account summary information will be displayed.
LastTransactions 0 1 LastTransactions The last transaction (payment or invoice) made with this element.
Summaries 0 1 SummaryList A list of balance summary and details summary data values retrieved from the database.

ElementFinderNumberRange

The subset of the list of elements to be retrieved. Use an extent of zero to list all elements.

Child elements

Name Minimum Maximum Type Description
Begin 1 1 int The position, counting from 0, of the first element in the list to retrieve.
Extent 1 1 int The number of elements to retrieve.

FindElementFilter

This element contains filter criteria for finding elements. Items left blank select all values.

Child elements

Name Minimum Maximum Type Description
NumberRange 1 1 ElementFinderNumberRange The subset of the list of elements to be retrieved. Use an extent of zero to list all elements.
CmpCode 1 1 typeGeneralCode The code of the company in which the element is maintained.
Level 0 1 typeElmLevel The element level to which the element you want to find belongs.
Code 0 1 typeElmCode The code of the element you want to find.
ShortName 0 1 typeShortNameB The short name of the element you want to find.
NameOrShortName 0 1 typeTextB The name or short name of the element you want to find.
ReturnNames 0 1 boolean Specifies that the element full name is displayed.
GroupCode 0 1 typeGeneralCodeB The element group to which the required element belongs.
CreditReference 0 1 typeCrRef The credit reference assigned to the element.
AddressName 0 1 typeAddressName The address name, as defined in the name and address list of the element.
Address1 0 1 typeAddress Line one of an address defined in the element's address list.
Address2 0 1 typeAddress Line two of an address defined in the element's address list.
PostCode 0 1 typePostCode The postcode defined in the element's address list.
Country 0 1 typeTextB The country defined in the element's address list.
AccNum 0 1 typeBankAccNum The bank account number of the element you want to find.
Tel 0 1 typeTel The telephone number defined in the element's address list.
Language 0 1 typeLanguageb The element language.
IBAN 0 1 typeIBANB The International Bank Account Number (IBAN) of the bank account.

elmKeyDataElement

Contains key information for each of the retrieved elements.

Child elements

Name Minimum Maximum Type Description

Code 1 1 typeElmCodeB The code of this element.
ShortName 1 1 typeShortNameB The abbreviated name of this element.
Name 1 1 typeNameB The full name of this element.
DefAccSumm 0 1 typeGeneralCode The default account summary of this element.
TemporaryElm 1 1 boolean Is this element a temporary supplier?
