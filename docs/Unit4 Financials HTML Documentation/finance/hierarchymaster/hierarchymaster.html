Child elements

Name Minimum Maximum Type Description
Code 1 1 typeCode The code for this hierarchy node.
HierarchyCode 1 1 typeCode The code for this hierarchy.
HierarchyLevel 1 1 typeCtHimLevel The hierarchy level at which the node exists.
GrpType 0 1 typeCtHimGrpType Specifies whether the leaf node represents an element group or an account group.
CmpCode 0 1 typeGeneralCodeB The code for the company in which the element group exists.
GrpCode 0 1 string The code for the element group or account group, whichever applies. The contents of GrpCode depends on GrpType as follows: him_grptype_accgrp - a value of type com:typeGeneralCode will be returned (account group) him_grptype_elmgrp - a value of type com:typeGeneralCode will be returned (element group) him_elm - a value of type com:typeElmCode will be returned (element code). This may come from the element master or the hierarchy master, depending on the 'use_element_name' setting on the hierarchy master.
