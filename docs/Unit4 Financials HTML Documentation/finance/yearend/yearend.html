Root Elements

Params

This document contains the parameters for the Year End process.

YearEndRequest

This service allows you to run and monitor the Year End process.

YearEndResponse

This is the response that is returned when you run and monitor the Year End process.

Params

This document contains the parameters for the Year End process.

Child elements

Name Minimum Maximum Type Description
CmpCode 1 1 typeGeneralCode The code of the company for which you want to run the year end process.
ProposedYearEnd 1 1 typeWord The company year for which you want to run the Year End process
YearEndDate 0 1 typeDateB The date for the Year End documents.
LimitSizeOfDocuments 1 1 boolean If TRUE, specifies that a Year End document will have a maximum size.
MaxAccountsPerDocument 0 1 typeULong The maximum number of lines that can be printed to a Year End document. This value must be in the range 1000 to 9999.
IncludeQuantityCalcs 1 1 boolean If TRUE, specifies that quantity values are brought forward in the Year End documents.
IncludeSummaryInLog 1 1 boolean If TRUE, specifies that the Year End log displays the Year End summary file.
IncludePLAccounts 1 1 boolean If TRUE, specifies that Year End posts documents to the Profit and Loss accounts.
IncludeZeroValueAccounts 1 1 boolean If TRUE, specifies that document lines that balance to zero are included in the Year End documents.
BSDocumentCode 1 1 typeGeneralCode The code that identifies the document master governing the Balance Sheet posting. This is a reversing document.
BSRetainedEarningsAccount 1 1 typeAcCode The code of the account that you have defined for profits. The net profit for the year is posted to this account. It cannot contain an umbrella element.
BSSuspenseAccount 0 1 typeAcCode The code that identifies the suspense account. This account is used to balance Year End documents when LimitSizeOfDocuments is TRUE.
PLDocumentCode 0 1 typeGeneralCode The code that identifies the document master governing the Profit and Loss posting.
PLSuspenseAccount 0 1 typeAcCode The code that identifies the suspense account. This account is used to balance Year End documents when LimitSizeOfDocuments is TRUE.
IsYearendStopped 0 1 typeBool Indicates whether the Year End process has been stopped by the user.
IsSummaryComplete 0 1 typeBool Indicates whether the Summary is complete.
MustBalanceCurrencies 1 1 MustBalanceCurrencies This element contains details of all currencies which must balance.
PLDocumentDescription 0 1 typeDescr The description to be used for the year end Profit & Loss document lines.
BSDocumentDescription 0 1 typeDescr The description to be used for the year end Balance Sheet document lines.
ReversingDocumentDescription 0 1 typeDescr The description to be used for the year end reversing document lines.
