Root Elements

BrowseLedgerRequest

This service allows you to browse summarised data for a selected range of accounts.

BrowseLedgerResponse

This is the response that is returned when you browse a ledger.

Filter

This element contains the criteria used to retrieve ledger details form the database.

Complex Types

FilterElement

This element holds the filter parameters for one column.

Child elements

Name Minimum Maximum Type Description
ColumnName 1 1 typeName The name of the column to which the filter parameters will apply.
OperatorID 1 1 typeCtVocabOp The operator used to define the values used as filter criteria.
From 1 1 typeGSLValue The first value in the selection range, or the value to be matched when using the equal to (=) operator.
To 0 1 typeGSLValue The last value in the selection range.
IncludeColumnsToRight 0 1 boolean If True, includes the sum of the specified column and all transaction-based columns to the right of this column.

FilterList

This element contains selection criteria used in addition to the criteria contained in the SelectFilter element, but the selection is based on columns in the ledger rather than selector vocabulary. Any column filter which can be expressed as a vocabulary-based filter, will be moved to the SelectFilter by the server to make the selection process more efficient.

Child elements

Name Minimum Maximum Type Description
Column 0 unbounded FilterElement This element holds the filter parameters for one column.

Child elements
