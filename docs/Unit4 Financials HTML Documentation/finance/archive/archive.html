http://www.coda.com/efinance/schemas/archive

Root Elements

ArchiveDocumentsInformation

This element holds information about archiving documents, and the state of the data during an Archive process.

ArchiveMonitor

This element holds information about the progress of the archive process.

ArchiveRequest

This service allows you to run and monitor Archive functions.

ArchiveResponse

This is the response that is returned when you run and monitor Archive functions.

Complex Types

ArchiveDateRange

If TRUE specifies that missing documents will be archived.

Child elements

Name Minimum Maximum Type Description
FromDate 1 1 typeDate The start date from which missing documents are selected for archiving.
ToDate 1 1 typeDate The end date from which missing documents are selected for archiving.

ArchiveDocumentsParams

Child elements

Name Minimum Maximum Type Description
ArchiveHeaderMasterKey 1 1 Key A generic key consisting of the company code and the archive header master code.
Selector 0 1 typeGeneralCode The selector master code used as the default when starting to archive documents.
SelectorVocabList 0 1 VocListData The list of data items in the selector master, which determine the documents to be archived.
IncludeArchives 1 1 boolean If TRUE specifies that old archive summary journals are to be removed along with other documents. This is only relevant when summary journals are not being created. Archive journals are not moved to the internal archive tables but are removed from the Books.
IncludeYearEnd 1 1 boolean If TRUE specifies that old year end journals are to be archived along with other documents. This is only relevant when summary journals are not being created.
MissingDates 0 1 ArchiveDateRange If TRUE specifies that missing documents will be archived.
Balances 1 1 typeCtARCBal Specifies whether Financials balances are archived, all balances are archived, or no balances are archived.
Year 1 1 short The year of the documents being archived.
FromPeriod 1 1 typePeriods The first period from which documents are being archived.
ToPeriod 1 1 typePeriods The last period from which documents are being archived.
Logging 1 1 typeCtARCLogType Specifies the type of log required: Full, Accepted documents only, Rejected documents only, or no log at all.
LogTitle 0 1 typeLogTitle The title of the log, if applicable.
ProceedOnBalanceWarning 0 1 typeBool If true, specifies that you want to proceed with Archive Balances, even if a warning about unarchived documents has been given.

BalanceWarning

This contains a warning that the Archive Balances process has found documents for that year which have not yet been archived.

Child elements

Name Minimum Maximum Type Description
Reason 1 1 Reason The reason why this warning message is sent.

ArchiveRecovery

Contains information on whether the Archive process has failed and needs to be recovered.

Child elements

Name Minimum Maximum Type Description
RecoveryNeeded 1 1 typeCtARCRecoveryNeeded States whether the process has crashed and needs to be recovered.
RecoveryPermitted 1 1 boolean If true, indicates that the Archive process should be restarted to clean up any errors in the data following a failure.

ArchiveDocumentsInformation

This element holds information about archiving documents, and the state of the data during an Archive process.

Child elements

Name Minimum Maximum Type Description
Params 1 1 ArchiveDocumentsParams This element contains the parameters for the archive process.
CurrentState 1 1 typeCtArchiveState The current status of the archive process.
ExternalArchive 1 1 boolean If TRUE specifies that this is an external archive.
HasJournals 1 1 boolean If TRUE specifies that archive journals will be produced.
DefaultMissingDates 0 1 ArchiveDateRange Contains a range of dates between which missing documents will be archived.
SingleYear 0 1 short The year for which documents are archived, if the archive is for a single year. Leave blank to include more than a single year.
ArchiveUserCode 0 1 typeGeneralCode The code of the user initiating the archive process.
Recovery 0 1 ArchiveRecovery Contains information on whether the Archive process has failed and needs to be recovered.

ArchiveLoadUnloadSelectionParams

This element holds the parameters for loading or unloading archive documents.

Child elements

Name Minimum Maximum Type Description
ArchiveHeaderMasterKey 1 1 Key A generic key consisting of the company code and the archive header master code.
Year 0 1 short The year of the archived documents being loaded or unloaded.
FromPeriod 1 1 typePeriods The first period for which you want to load or unload archived documents.
ToPeriod 1 1 typePeriods The last period for which you want to load or unload archived documents.

FromDocumentNumber 0 1 typeDocNum The first document number in the range of archived documents that you want to load or unload.

ArchiveRunTime

This element contains the accumulated run time since the archive process began.

Child elements

Name Minimum Maximum Type Description
Days 1 1 short The number of days the archive process has been running so far.
Hours 1 1 short The number of hours the archive process has been running so far.
Minutes 1 1 short The number of minutes the archive process has been running so far.

ArchiveMonitor

This element holds information about the progress of the archive process.

Child elements

Name Minimum Maximum Type Description
ArchiveHeaderMasterKey 1 1 Key A generic key consisting of the company code and the archive header master code.
ExternalArchive 1 1 boolean If TRUE specifies that this is an external archive.
FromPeriod 1 1 typePeriod The first year/period for which documents are being archived.
ToPeriod 1 1 typePeriod The last year/period for which documents are being archived.
LastDocumentCode 1 1 typeGeneralCode The code of the most recent document to be processed.
LastDocumentNumber 1 1 typeDocNum The number of the most recent document to be processed.
LastDocumentCount 1 1 integer The number of documents archived so far.
LastJournalCode 1 1 typeGeneralCode The code of the most recent journal written.
LastJournalNumber 1 1 typeDocNum The number of the most recent journal written.
LastJournalCount 1 1 integer The number of journals written so far.
RunTime 1 1 ArchiveRunTime This element contains the accumulated run time since the archive process began.
CurrentState 1 1 typeCtArchiveState The current status of the archive process, for example 'Selecting documents'.
Paused 1 1 boolean If TRUE indicates that the archive process has paused.
LogID 0 1 int The identifier for the log containing details of the Archive process.

ArchiveRequest

The ArchiveRequest type has ServiceRequest as its base type.

This service allows you to run and monitor Archive functions.

Child elements

Name Minimum Maximum Type Description
StartArchivingDocuments 0 1 StartArchivingDocumentsRequestVerb Starts the Archive Documents process.
GetArchiveDocumentsInformation 0 1 GetArchiveDocumentsInformationRequestVerb Retrieves default parameters and current status for the Archive Documents process (prior to starting Archive Documents).
Abort 0 1 progressAbortRequestVerb Stops the Archive process, which cannot then be restarted.
Pause 0 1 progressPauseRequestVerb Pauses the Archive process.
Resume 0 1 progressResumeRequestVerb Resumes an Archive process which you have paused.
GetProgress 0 1 progressGetRequestVerb Retrieves information on the progress of the Archive process. You need the key, or special reference, given to you for the process you started.
Restart 0 1 RestartRequestVerb Restarts an Archive process which has failed. You need to send this request if the response in the Recovery section of GetArchiveDocumentsInformation advises that the process has failed.
