http://www.coda.com/efinance/schemas/auditdocstatus

Root Elements

AuditDocStatusRequest

This service allows you to recover documents from the database.

AuditDocStatusResponse

This is the response that is returned when you recover documents from the database.

ProgressStatistics

This element contains statistical information about the progress of the document recovery process.

StartParams

This element contains the parameters used to select the documents to recover. This process is performed using the Document Status Toolbox function.

StartParams

This element contains the parameters used to select the documents to recover. This process is performed using the Document Status Toolbox function.

Child elements

Name Minimum Maximum Type Description
DocCode 1 1 typeGeneralCode The code of the document to recover.
DocNumber 0 1 typeDocNum The number of the document to recover.
FromDate 0 1 typeHSDate The first date in the date range from which you want to select unusable documents.
ToDate 0 1 typeHSDate The last date in the date range from which you want to select unusable documents. This date must be equal to or later than the date entered for the start of the range, and at least one day before the current date.
LogTitle 0 1 typeLogTitle The title of the log generated by the process.
BatchMode 1 1 boolean Specifies whether this document toolbox process will recover multiple documents.
TableText 1 1 typeLogTitle A text description giving the reason why you are running the document recovery process.
