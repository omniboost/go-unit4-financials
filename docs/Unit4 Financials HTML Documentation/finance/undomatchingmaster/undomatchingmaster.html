http://www.coda.com/efinance/schemas/undomatchingmaster

Root Elements

UndoMatchingMaster

This element holds an undo matching master.

UndoMatchingMasterRequest

This service allows you to maintain undo matching masters.

UndoMatchingMasterResponse

This is the response that is returned when you maintain undo matching masters.

Complex Types

MatchType

The types of document lines to unmatch.

Child elements

Name Minimum Maximum Type Description
Paid 1 1 boolean Specifies that document lines with a payment status of Available (before being matched) are unmatched.
Deferred 1 1 boolean Specifies that document lines with a payment status of Draft Available (before being matched) are unmatched.
Suppressed 1 1 boolean Specifies that document lines with a payment status of Suppressed Paid (before being matched) are unmatched.
CancelMatched 1 1 boolean Specifies that individual Suppressed Paid lines are unmatched.

UndoMatchingMaster

This element holds an undo matching master.

Child elements

Name Minimum Maximum Type Description
TimeStamp 1 1 short The TimeStamp value for this undo matching master in the database.
CmpCode 1 1 typeGeneralCode The code for the company in which the undo matching master is being maintained.
Code 1 1 typeGeneralCode The code for this undo matching master.
Name 1 1 typeName The full name for this undo matching master.
ShortName 1 1 typeShortNameB The abbreviated name for this undo matching master.
UndoType 1 1 typeCtUndoMatchType Specifies whether this master is used to undo matching references or payment numbers.
MatchType 1 1 MatchType The types of document lines to unmatch.
SelCode 1 1 typeGeneralCode The code of the selector master that selects the document lines to be unmatched.
PresCode 1 1 typeGeneralCode The code of the presenter master used to display the selected document lines.
CancelPostings 1 1 boolean Specifies whether disperse and matching postings that were made during the original match are reversed. When TRUE, you must specify a cancelling code for CancelDocCode.
CancelDocCode 0 1 typeGeneralCode The code of the cancelling document to use when postings are reversed by undo matching.
EnableLogging 1 1 boolean Specifies whether a log of the undone match(es) is created.
