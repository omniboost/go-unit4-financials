UndoMatchingParams

This element contains the parameters which can be specified when performing undomatching.

Child elements

Name Minimum Maximum Type Description
CmpCode 1 1 typeGeneralCode The code for the company in which undo matching is being performed.
UndoMatchingMasterCode 1 1 typeGeneralCode The code of the undomatching master being used.
UndoType 1 1 typeCtUndoMatchType Indicates whether matching references or payment numbers will be undone.
MatchType 1 1 MatchType Indicates the types of document lines that will be unmatched.
UndoMatchingDate 1 1 typeDate The date on which undo matching is performed. This defaults to today's date.
Period 1 1 typePeriod The year and period in which undo matching is performed. These must be valid for the undo matching date. If cancelling documents are generated, they are posted to this year and period.
LogTitle 0 1 typeLogTitle The file to which the undo matching log is written. This is only available if the undo matching master has logging enabled.

MatchKey

Child elements

Name Minimum Maximum Type Description

ElmCode 0 1 typeElmCode The code of the element for which you want to undo matches.
PaymentNum 0 1 int The payment number of the match to be undone.
MatchRef 0 1 int The matching reference of the match to be undone.
