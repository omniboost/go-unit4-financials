Root Elements

InputTemplate

This element holds an input template master.

InputTemplateRequest

This service allows you to maintain input template masters.

InputTemplateResponse

This is the response that is returned when you maintain input template masters.

Simple Types

The range of values that are valid for the GeneratedFieldsPerRow element.

Minimum value 3
Maximum value 8
