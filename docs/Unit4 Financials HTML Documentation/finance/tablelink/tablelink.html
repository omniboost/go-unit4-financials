DTLStartParams

This element contains the parameters for the Table Link document transfer process.

Child elements

Name Minimum Maximum Type Description
CmpCode 1 1 typeGeneralCode The company code.
DocCode 0 1 typeGeneralCode The document code.
DocStartNumber 0 1 typeDocNum The number of the first document in the selection range.
DocEndNumber 0 1 typeDocNum The number of the last document in the selection range.
LinkCode 0 1 typeGeneralCode The linkcode entry in the oas_linkhead table that identifies a document or group of documents.
TreatWarningsAsErrors 1 1 boolean If true, specifies that warnings should be handled as if they were errors.
StopRunOnError 1 1 boolean If true, stops the Table Link process if an error occurs transferring a document.
ReportMode 1 1 typeCtLinkReportMode The type of log to generate.
Destination 1 1 typeCtDocDest The destination for the transferred documents.
LogTitle 1 1 typeTitle A title for the log generated when you run the Table Link process.
BreakTriangulation 1 1 boolean If true, specifies that triangulation rules should be ignored.
LogOnlyFailures 1 1 boolean If true, specifies that the log will contain only failures and end-of-run statistics.

DLNStartParams

This element contains the parameters for the Table Link document line comments or document line description transfer process.

Name Minimum Maximum Type Description

DocNumTo 0 1 typeDocNum The number of the last document in the selection range.
LinkCode 0 1 typeGeneralCode The linkcode entry in the oas_linkcomment table (comments) or in the oas_linkdesc table (line descriptions).
LogTitle 1 1 typeTitle A title for the log generated when you run the Table Link process.
