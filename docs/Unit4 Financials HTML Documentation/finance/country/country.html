CountryMaster

This element holds a country master.

Child elements

Name Minimum Maximum Type Description
Code 1 1 typeGeneralCode The code for this country master.
Name 1 1 typeName The full name for the country master.
TimeStamp 1 1 short The TimeStamp value for this country master in the database.
AlternativeCode 1 1 typeGeneralCode The code that identifies the country in electronic transactions. For example, this could be a two-character ISO code of the country, or one of the other coding schemes.
Language 1 1 typeLanguage The code that identifies the language that will be used as the default for all new elements that specify this country in their address.
PostCode 1 1 string If true, specifies that the format of this country's post codes will be validated by a user-defined routine.
Bank 1 1 string If true, specifies that the format of this country's bank account numbers and sort codes will be validated by a user-defined routine.
Tax 1 1 string If true, specifies that the format of this country's tax numbers will be validated by a user-defined routine.
IBAN 1 1 string If true, specifies that the format of this country's IBANs and SWIFT addresses will be validated by a user-defined routine.
BankSortCodeRequired 1 1 boolean If true, specifies that the bank sort code fields on the bank and element masters are mandatory.
BankAccountNameRequired 1 1 boolean If true, specifies that the bank account name fields on the bank and element masters are mandatory.
BankReferenceRequired 1 1 boolean If true, specifies that the bank reference fields on the bank and element masters are mandatory.
BankIBANRequired 1 1 boolean If true, specifies that the IBAN field on the element master's bank details is mandatory.
PostCodeExtension 0 1 ExtensionRef The code of the user extension master that will be used to validate or reformat the layout of this country's post codes in the element master and bank addresses.
BankExtension 0 1 ExtensionRef The code of the user extension master that will be used to validate or reformat the layout of this country's bank account numbers and sort codes in the element and bank masters.
TaxExtension 0 1 ExtensionRef The code of the user extension master that will be used to validate or reformat the layout of this country's tax numbers in the element and company masters.
IBANExtension 0 1 ExtensionRef The code of the user extension master that will be used to validate or reformat the layout of this country's IBANs and SWIFT addresses on the bank master and on element bank details.
