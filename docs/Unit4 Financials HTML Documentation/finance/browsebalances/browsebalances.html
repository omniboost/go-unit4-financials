Root Elements

BrowseBalancesRequest

This service allows you to retrieve browse balance information.

BrowseBalancesResponse

This is the response that is returned when you browse balance information.

Complex Types

This element holds the selection vocabulary used to retrieve the balance information from the database.

Child elements

Name Minimum Maximum Type Description
UserFilter 0 1 VocListData The filter criteria defined by the user. If the same vocabulary item appears in both the UserFilter and ApplicationFilter elements, the value held by the ApplicationFilter element is used to select the data.

FlexiFieldFilter 0 1 FlexiFieldVocListData The flexi-field selection criteria.

ColInfo

This element holds information about one column contained in the Browse Balances report.

Child elements

Name Minimum Maximum Type Description
ID 1 1 typeBBQColumnID The name specified for the column in the browse balance master. It is the name displayed on the column tab, and is different to column Title.
DataType 1 1 typeCtBBQColDataType The type of data displayed in the column. This can be BBQ-NUMBER, BBQ-TEXT or BBQ-CODE-AND-SNAME.
Title 0 1 typeTextAnyB The title displayed at the head of the column.
Type 1 1 typeCtBBQColumnType Identifies the column type. The available column types are: Balance, Master Information, Hierarchy or Calculation.
VocabID 0 1 typeCtVocabEnum The vocabulary data displayed on the column, for example, Element 1.
Width 1 1 int The width of the column.
Decimals 0 1 int The number of decimal places to which the balance is displayed. The available values are Default and the numbers 0 to 9. If you select a number, the values in the column are rounded to that number of decimals places.
LinkedTo 0 1 typeTextAnyB The name of the column to which the current column is linked.
ComparativeCol 0 1 boolean Indicates whether this column is linked to another column so that the data can be compared.
AccumulateSel 0 1 boolean Indicates whether the values in this column are accumulated in period order so that cumulative balance figures can be shown.
IncludeBroughtForwards 0 1 boolean Indicates whether balances which have been brought forward are included in the accumulation, in addition to the selected balances.
PreCode 0 1 typeGeneralCodeB The code for the presenter master to use when drilling to Browse Details.
CmpCode 0 1 typeGeneralCodeB The code for the company used to determine the presenter master to use when drilling to Browse Details.
Calculation 0 1 typeTextAnyUpperB The formula used to calculate the column values.
CommitmentColumn 0 1 boolean Indicates whether this column contains commitment balances. These are the balances for commitment accounting in Procurement.
Alignment 0 1 typeCtJustification The alignment of the values in this column. The available values are Left, Right and Centre.
Bold 0 1 boolean Indicates whether the values in this column are displayed in bold.
Italic 0 1 boolean Indicates whether the values in this column are displayed in italics.
Underline 0 1 boolean Indicates whether the values in this column are underlined.
ReportTotalSign 0 1 string Specifies the values to use when calculating the total for this report. The available values are: Original values, Signed values and Original values reverse sign.
ReverseSign 0 1 string Specifies whether the sign (+ or -) on column values is reversed.
CalculationDefaultDecimals 0 1 int The default number of decimal places for a calculation type column.
FlexiFieldVocabDataType 0 1 typeCtFlexiFieldDataTypes The data type of a flexi-field value.
FlexiFieldDecimalPlaces 0 1 short The number of decimal places in a flexi-field value.

ColInfoList

This element holds information about the columns contained in the Browse Balances report. These columns are set up in the browse balance master.

Child elements

Name Minimum Maximum Type Description
ColInfo 0 unbounded ColInfo This element holds information about one column contained in the Browse Balances report.

Cell

The title for one column.

Attributes

Name Type Use Default Description
currcode string optional The currency code for the value in a balance column cell.
dividebyzero string optional Indicates whether the calculation performs a division by zero.
footer string optional The footer text for accounts under this node in a hierarchy.
header string optional Alternative header text for accounts under this node in a hierarchy.
mixedcurr string optional Contains the string 'True', indicating that a balance column cell contains a number which is the sum of mixed currency values.
originalvalue string optional The original balance value, before reverse sign settings are applied.
overflow string optional Indicates whether the result of the calculation exceeds the maximum size for a calculated column.
subtotalsign string optional Indicates the values to use when calculating the subtotal for this node. The available values are: Original values, Signed values and Original values reverse sign.
vocabdata string optional The vocabulary data used to identify the contents of a cell.
