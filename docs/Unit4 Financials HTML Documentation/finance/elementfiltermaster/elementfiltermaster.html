Root Elements

ElementFilterMasterRequest

This service allows you to maintain element filter masters.

ElementFilterMasterResponse

This is the response that is returned when you maintain element filter masters.

Child elements

Name Minimum Maximum Type Description
Logical 1 1 typeCtAccessLog A logical operator, either AND or OR.
LBracket 1 1 typeCtLeftBracketVocab The number of opening brackets to insert at this point (for grouping purposes).

Function 1 1 typeCtAccessOp A number identifying the relational operator that specifies the kind of comparison made between element codes matching the Vocab item, and the supplied Value.
Value 1 1 string A pattern value that is compared to element codes matching the Vocab item. You can use wildcards only with the LIKE and NOTLIKE functions.
TBracket 1 1 typeCtRightBracketVocab The number of closing brackets to insert at this point (for grouping purposes).
