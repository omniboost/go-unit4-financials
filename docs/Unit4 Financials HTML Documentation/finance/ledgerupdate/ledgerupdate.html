Root Elements

IsOneRunningStatistics

This element contains statistical information about whether a ledger update process is running and the parameters used to start the last, or currently running, update.

LedgerUpdateRequest

This service allows you to update ledger information.

LedgerUpdateResponse

This is the response that is returned when you update ledger information.

ProgressStatistics

This element contains statistical information about the progress of this ledger update.

StartParams

This element contains the parameters used to select the ledgers to update.

Complex Types

StartParams

This element contains the parameters used to select the ledgers to update.

Child elements

Name Minimum Maximum Type Description
UpdateMode 1 1 typeCtUpdateMode Should the process continually update or stop after one update? The available values are Single update (ledger information is updated only once) or Continuous update (ledger information is updated repeatedly at intervals specified in the IdleInterval field).
IdleInterval 0 1 typeULong The interval in hours between each ledger update. This option is available only when the UpdateMode is Continuous.
AllCompanies 1 1 boolean Specifies whether you want to update all ledgers in all the companies to which you have access.
LedgerSelection 0 1 typeGeneralCode The code of the ledger master you want to update. Leave this field blank to select all ledger masters for the current company.
ReferenceDate 0 1 typeHSDateB The date for which the selected ledger master(s) will be updated. This can be a hard date or a soft date, or can be left blank to use the reference date specified on the individual ledger master(s). If the reference date on a ledger master is blank, and ReferenceDate is left blank, the ledger will be updated for today's date.

ProgressStatistics

This element contains statistical information about the progress of this ledger update.

Child elements

Name Minimum Maximum Type Description
OverallElapsedDays 1 1 short The time elapsed in days since the start of the ledger update process (this includes all current update cycles).
OverallElapsedHours 1 1 short The time elapsed in hours since the start of the ledger update process (this includes all current update cycles).
OverallElapsedMins 1 1 short The time elapsed in minutes since the start of the ledger update process (this includes all current update cycles).
IdleHoursLeft 0 1 short The time remaining in hours until the next update cycle begins. This information is displayed only when the current state of the update process is Idle.
IdleMinsLeft 0 1 short The time remaining in minutes until the next update cycle begins. This information is displayed only when the current state of the update process is Idle.
CyclesCompleted 1 1 short The number of update cycles completed so far in the ledger update process.
CycleStartDate 0 1 typeDate The date on which this update cycle started.
CycleStartTime 0 1 typeTime The time at which this update cycle started.
CycleElapsedDays 0 1 short The time elapsed in days since the start of this update cycle.
CycleElapsedHours 0 1 short The time elapsed in hours since the start of this update cycle.
CycleElapsedMins 0 1 short The time elapsed in minutes since the start of this update cycle.
TotalCompanys 0 1 short The total number of companies for which ledgers are being updated.
CurrentCompany 0 1 short The company for which ledger information is being updated.
CurrentCompanyCode 0 1 typeGeneralCode The code of the company for which ledger information is being updated.
TotalLedgers 0 1 short The number of ledgers in the update of the current company.
CurrentLedger 0 1 short The ledger master currently being updated.
CurrentLedgerCode 0 1 typeGeneralCode The code for the ledger master currently being updated.
DetailsSelected 1 1 typeULong The number of document lines selected for this ledger update cycle.
DetailsProcessed 1 1 typeULong The number of detail lines processed so far for this ledger update cycle.
OverallState 1 1 typeCtOneOfN The current state of the Ledger Maintenance process. The available values are: None, Processing, Idle, Aborting, Finishing, Finished.
CycleState 1 1 typeCtOneOfN The state of the current update cycle. The available values are: None, Selecting, Building, Updating and Writing statistics.
ReasonMessage 0 1 typeText Displays information about the ledger update cycle.

IsOneRunningStatistics

This element contains statistical information about whether a ledger update process is running and the parameters used to start the last, or currently running, update.

Child elements

Name Minimum Maximum Type Description
IsRunning 1 1 boolean Specifies whether a ledger update process is currently running.
OverallStartDate 0 1 typeDate The date on which the ledger update process started.
OverallStartTime 0 1 typeTime The time at which the ledger update process started.
UpdateMode 1 1 typeCtUpdateMode Should the process continually update or stop after one update? The available values are Single update (ledger information is updated only once) or Continuous update (ledger information is updated repeatedly at intervals specified in the IdleInterval field).
IdleInterval 0 1 typeULong The interval in hours between each ledger update. This option is available only when the UpdateMode is Continuous.
CompanySelection 1 1 typeGeneralCode The code of the company for which a ledger update process was last run, or is currently running. A blank field indicates that ledger information is being updated for all the companies to which you have access.
LedgerSelection 0 1 typeGeneralCode The code for the ledger master to update. Leave this field blank to select all ledger masters for the current company.
