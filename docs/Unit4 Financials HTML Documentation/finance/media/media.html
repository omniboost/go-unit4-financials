Root Elements

MediaMaster

This element contains a media master.

MediaReqKeys

This is a media master key used when requesting a list of media masters. It incorporates the company within which the media master exists and a value for the maximum number of items to return.

MediaRequest

This service allows you to maintain media masters.

MediaResponse

This is the response that is returned when you maintain media masters.

Complex Types

MediaMaster

This element contains a media master.

Child elements

Name Minimum Maximum Type Description
TimeStamp 1 1 short The TimeStamp value for this media master in the database.
CmpCode 1 1 typeGeneralCodeB The code of the company in which the media master exists.
Code 1 1 typeGeneralCode The code of the media master.
Name 1 1 typeName The full name for the media master.
ShortName 1 1 typeShortNameB The abbreviated name for the media master.
PayMethod 1 1 typeCtPayMedia The method by which the payment will be made, either electronic or paper.
ElecMethod 1 1 typeCtElectronicMethod For electronic methods, specifies whether the payment is a Direct Debit or Credit Transfer.
MaxTransaction 1 1 typeMoney The maximum value of an individual payable instrument. Valid for an electronic payment.
MaxTotal 1 1 typeMoney The maximum of the sum of all the payable instrument values. Valid for an electronic payment.
LinesPerPage 1 1 short The number of lines to be printed per page of the remittance advice. Valid for an electronic payment.
AutoNumber 1 1 boolean If TRUE, specifies that the documents are numbered automatically at the appropriate time. Valid for an electronic payment.
NoSummarization 1 1 boolean If TRUE, specifies that the element summary and split levels are ignored when being paid by the current medium.
PreNum 1 1 boolean If TRUE, specifies that documents will be numbered before the production stage of a payment or collection process.
SeperateChequeAndRemittance 1 1 boolean If TRUE, specifies that separate stationery for printing cheques and remittance notes will be used.

MediaReqKeys

This is a media master key used when requesting a list of media masters. It incorporates the company within which the media master exists and a value for the maximum number of items to return.

Child elements

Name Minimum Maximum Type Description
MaxKeys 1 1 int The maximum number of items to be returned.
Key 0 1 Key A key consisting of a company code and a media master code.
WantElectronic 1 1 boolean Specifies whether media masters with an electronic payment method should be included in the list.
WantPaper 1 1 boolean Specifies whether media masters with a paper payment method should be included in the list.
ElmKey 0 1 elmFullKey An element key that is used to filter the list of media masters returned. The list contains only the media masters appropriate to this element.
