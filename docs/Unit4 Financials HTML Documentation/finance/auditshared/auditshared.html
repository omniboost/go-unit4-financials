This service allows you to audit shared masters.

This is the response that is returned when you audit shared masters.

This element contains statistical information about the progress of the audit.

This element contains the parameters used to select the masters to audit.

StartParams

This element contains the parameters used to select the masters to audit.

Child elements

Name Minimum Maximum Type Description
LogTitle 1 1 typeLogTitle The title of the log that will be generated by the audit process.
ProvCmpCode 1 1 typeGeneralCode The code of the provider company for which you want to run the audit.
DepCmpCode 0 1 typeGeneralCode The code of the dependent company for which you want to run the audit. Specify a single company or enter the wildcard * to audit all companies dependent on the provider company.
AllDepCmpCodes 1 1 boolean Indicates whether the audit will run for all the companies dependent on the provider company.

ProgressStatistics

This element contains statistical information about the progress of the audit.

Child elements

Name Minimum Maximum Type Description
MastersRead 1 1 typeULong The number of masters read so far by the audit process.
ErrorCount 1 1 typeULong The number of errors detected by the audit process.
Status 1 1 typeCtAdsProgressStatus The current status of the audit process.
ElapsedTime 1 1 ElapsedTime The time that has elapsed in days, hours and minutes since the start of the audit process.
