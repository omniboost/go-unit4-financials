Root Elements

AccountRequest

This service allows you to validate account codes.

AccountResponse

This is the response that is returned when you validate account codes.

ChkAccCodeData

This element holds details of the account code you want to validate and the type of validation you want to perform.

Transaction

This element holds a financial document.

TransactionRequest

This service allows you to retrieve documents (transactions) from the Financials database.

TransactionResponse

This is the response that is returned when you retrieve documents (transactions) from the Financials database.
