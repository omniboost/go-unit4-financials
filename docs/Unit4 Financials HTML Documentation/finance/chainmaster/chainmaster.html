http://www.coda.com/efinance/schemas/chainmaster

Root Elements

ChainMaster

This element holds a chain master.

ChainMasterRequest

This service allows you to maintain chain masters.

ChainMasterResponse

This is the response that is returned when you maintain chain masters.

Complex Types

Master

Child elements

Name Minimum Maximum Type Description
TimeStamp 1 1 short The TimeStamp value for this chain master in the database.
CmpCode 1 1 typeGeneralCodeB The code of the company in which the chain master exists.
Code 1 1 typeGeneralCode The code for this chain master.
Name 1 1 typeName The full name for this chain master.
ShortName 1 1 typeShortNameB The abbreviated name for this chain master.
ContinueOnFail 1 1 boolean If TRUE, specifies that the chain runs the remaining rules if one rule fails.
DocumentDate 0 1 typeDate The posting date for the allocation document.
DocumentPeriod 1 1 typePeriod The year and period for posting, which will apply to the allocation document. You can use an explicit or soft period.
ReversingDocumentDate 0 1 typeDate The date of the reversing document, if applicable.
ReversingDocumentPeriod 1 1 typePeriod The year and period to which the reversing document will be posted, if applicable.
BalancePeriod 1 1 typePeriod The year and period which you want to be updated, if single period update is set in the rule master. If multi-period update is set, this override is not applied.
DocumentLineDescription 1 1 typeTextAnyB A description of up to 36 characters which will be applied to each line of the document, overriding those specified in the individual rule masters.
ApplyOverrides 1 1 boolean If TRUE specifies that values set in the previous fields (dates, periods, document line descriptions) will override those set in the corresponding fields in the individual rule masters.

Name Minimum Maximum Type Description

RuleList

The list of rules which you want this chain to run. All the rules named within this element should be of the same rule type: either allocation rules or revaluation rules.
