Root Elements

SelectorMaster

This element holds a selector master.

SelectorMasterRequest

This service allows you to maintain selector masters.

SelectorMasterResponse

This is the response that is returned when you maintain selector masters.

SelectorType

This element contains information which determines the selector type.

Child elements

Name Minimum Maximum Type Description

SourceModeSelectorType

A 32-bit number representing the data class and the data location.

Child elements

Name Minimum Maximum Type Description

Intray 0 1 boolean Documents located on the Financials Intray will be retrieved.
Archive 0 1 boolean Documents located in the internal archive tables will be retrieved.
SourceMode 0 1 boolean If TRUE, then only selector masters with exactly the given combination of data locations will be returned. If FALSE, selector masters will be returned if the specified data locations are present on it but also those with additional locations specified.
