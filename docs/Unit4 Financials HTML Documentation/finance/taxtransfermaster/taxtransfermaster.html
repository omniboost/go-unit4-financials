Root Elements

TaxTransferMaster

This element holds a tax transfer master.

TaxTransferMasterRequest

This service allows you to retrieve tax transfer masters.

TaxTransferMasterResponse

This is the response that is returned when you maintain tax transfer masters.

Complex Types

TaxTransferMaster

This element holds a tax transfer master.

Child elements

Name Minimum Maximum Type Description
CmpCode 1 1 typeGeneralCode The code for the company in which the tax transfer master is being maintained.
Code 1 1 typeGeneralCode The code for this tax transfer master.
TimeStamp 0 1 short The TimeStamp value for this tax transfer master in the database.
Name 1 1 typeName The full name for this tax transfer master.
ShortName 0 1 typeShortName The abbreviated name for this tax transfer master.
DeferredTaxFromDate 1 1 typeHSDate When transferring deferred VAT the process is initiated by a date range. All lines whose paid date or undo-match date is later than this date, and earlier than the date set in the DeferredTaxToDate element will be selected for processing.
DeferredTaxToDate 1 1 typeHSDate When transferring deferred VAT the process is initiated by a date range. All lines whose paid date or undo-match date is earlier than this date, and later than the date set in the DeferredTaxFromDate element will be selected for processing.
SelCode 0 1 typeGeneralCode The code of the selector master which the transfer process uses.
TransferDocCode 1 1 typeGeneralCode The code of the document master used to create the transfer documents.
TransferDocDate 1 1 typeHSDate The date of the transfer documents.
TransferDocPeriod 1 1 typeSoftPeriod The year and period of the transfer documents.
ProtectPeriodAndDate 1 1 boolean Specifies that users cannot edit the transfer document date and period at run time.
Description 0 1 typeDescr The default description for the transfer document lines.
