ExpertParameters

This element contains all the expert parameters in the interest master.

Child elements

Name Minimum Maximum Type Description
InterestRate 1 1 typeMoney Specifies the percentage interest rate.
InterestRateFrequency 1 1 typeCtCimInterestFrequency Specifies whether the stated interest rate is daily or annual.
ExtraDays 1 1 int The number of days for which you want to charge interest in addition to the number of days the settlement is overdue.
SingleCharge 1 1 typeMoney The amount, in your home currency, that will be added to the interest charge when an invoice first becomes due.
RecurrentCharge 1 1 typeMoney The amount, in your home currency, that will be added to the interest charge on the first and subsequent charging periods.
StartDate 1 1 typeHSDate The start of the charging period as either an explicit or a soft date.
EndDate 1 1 typeHSDate The end of the charging period as either an explicit or a soft date.
GracePeriod 1 1 int The number of days that an invoice may be overdue before interest is added to it.
RelevantDate 1 1 typeCtCimRelevantDate Specifies which date is the relevant charging date. Relevant dates are: Due date, Value date or Document date.
ChargeWhenFirstDue 1 1 typeCtCimPartialCharge Specifies which charging period will be used when the first invoice is due. Charging periods are: None and Partial.
ChargeWhenPaid 1 1 typeCtCimPartialCharge Specifies which charging period will be used when the invoice is paid. Charging periods are: None and Partial.

FormulaLineList

A list of lines contained in the formula.

Child elements

Name Minimum Maximum Type Description

InterestMaster

This element contains the settings for the interest master.

Child elements

Name Minimum Maximum Type Description
CmpCode 1 1 typeGeneralCode The code for the company in which you created this interest master.
TimeStamp 1 1 short The TimeStamp value for this interest master in the database.
Code 1 1 typeGeneralCode The code for this interest master.
Name 1 1 typeName The full name for this interest master.
ShortName 1 1 typeShortNameB The abbreviated name for this interest master.
ElementLevel 1 1 typeElmLevel The level of the element which, in conjunction with the contents of the ChargeCurrency element may specify the currency in which the interest will be charged.
ChargeCurrency 1 1 typeCtCimChargeCur The currency type that will be used to calculate the interest charge and to set the currency of the posting document. The charge currency options are: Home, Dual, Document and Element.
SelectorCode 1 1 typeGeneralCode The required selector code.
PresenterCode 1 1 typeGeneralCode The presenter code that you want to use for Browse.
ChargesDefaultHeld 1 1 boolean Specifies that entries will be held from the interest charge at run time.
PostingAccount 1 1 typeAcCode Specifies the account that the credit line will be posted to.
ChargeLevel 1 1 typeCtCimChargeLevel Specifies the type of charge level used with this master. The charge levels are: Account, Summary, and Transaction.
Destination 1 1 typeCtCimDestination Specifies where documents are posted. The destinations are Intray, Books or Books + Intray.
DocumentCode 1 1 typeGeneralCode The code that identifies the document master.
PostingYearPeriod 1 1 typeSoftPeriod The year and period to which the document will be posted specified as either an explicit or a soft year/period.
DocumentDate 1 1 typeHSDate The date on the document specified as either an explicit or a soft date.
DocumentDescription 1 1 typeTextB A brief description of the document.
ProtectDatesAndPeriod 1 1 boolean Specifies that the DocumentDate and PostingYearPeriod elements cannot be overwritten at run time.
Formula 1 1 FormulaLineList The formula specified for this interest master.
CreateInterestInvoice 0 1 boolean This element is obsolete.
