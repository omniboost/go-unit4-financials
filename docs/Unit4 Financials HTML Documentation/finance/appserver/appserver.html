http://www.coda.com/efinance/schemas/appserver

Root Elements

AppServerRequest

This service allows you to request environment and user company information from the server.

AppServerResponse

This is the response that is returned when you request environment and user company information from the server.

Environment

This element contains information about the application server's environment. The information returned depends upon the parameters used to obtain the authentication token sent with the Request.

SoftToHardDateData

This element contains details of a soft date for conversion to a hard date.

UserCmpsListKey

Contains key information for specifying the maximum number of companies to be returned by the application server.

Child elements

Name Minimum Maximum Type Description
