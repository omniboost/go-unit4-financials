RevaluationRequirements

This element contains the parameters for running a Currency Revaluation rule.

Child elements

Name Minimum Maximum Type Description
RuleCode 1 1 typeGeneralCode The code of the revaluation rule master you want to use.
NeedRateDate 1 1 boolean If TRUE, specifies that you will use the currency exchange rate on a specified date.
NeedForeignRate 1 1 boolean If TRUE, specifies that revaluation will be based on a foreign currency.
NeedForeignParentRate 1 1 boolean If TRUE, specifies that revaluation will be based on the rate of a foreign currency's parent currency, for example the euro.
NeedDualRate 1 1 boolean If TRUE, specifies that revaluation will be based on the company's dual currency.
NeedDualParentRate 1 1 boolean If TRUE, specifies that revaluation will be based on the dual currency's parent currency.
ForeignCurCode 1 1 typeGeneralCodeB The code of the foreign currency on which revaluation will be based.
DualCurCode 1 1 typeGeneralCodeB The code of the dual currency on which revaluation will be based.
ParentCurCode 1 1 typeGeneralCodeB The code of the parent currency on which revaluation will be based.

RevaluationRequirementsList

Child elements

Name Minimum Maximum Type Description

Child elements

Name Minimum Maximum Type Description

ForeignParentRate 0 1 typeRate The exchange rate of the foreign currency's parent currency, which you want to use.
DualRate 0 1 typeRate The exchange rate of the dual currency which you want to use.
DualParentRate 0 1 typeRate The exchange rate of the dual currency's parent currency, which you want to use.
