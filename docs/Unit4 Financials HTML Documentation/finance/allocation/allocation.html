http://www.coda.com/efinance/schemas/allocation

Root Elements

AllocationRequest

This service allows you to run and monitor Allocation.

AllocationResponse

This is the response that is returned when you run and monitor Allocation.

Complex Types

RuleParams

This element contains the requirements for running an allocation rule.

Child elements

Name Minimum Maximum Type Description
CmpCode 1 1 typeGeneralCode The code of the company in which you will run Allocation.
RuleCode 1 1 typeGeneralCode The code of the allocation rule master you want to use.
LogTitle 1 1 typeLogTitle The title of the log in which you want the details of the allocation to be recorded.

ChainParams

This element contains the requirements for running an allocation chain.

Child elements

Name Minimum Maximum Type Description
CmpCode 1 1 typeGeneralCode The code of the company in which you will run Allocation.
ChainCode 1 1 typeGeneralCode The code of the chain master which you want to use.
FirstRuleIndex 1 1 int The code of the rule at which you want the chain to start, if you do not want it to start at the beginning.
LogTitle 1 1 typeLogTitle The title of the log in which you want the details of the allocation to be recorded.

AllocationRequirements

Contains the requirements for an Allocation run.

Child elements

Name Minimum Maximum Type Description
RuleCode 1 1 typeGeneralCode The code of the rule master you want to use.
NeedFixedHomeValue 1 1 boolean If TRUE, specifies that a fixed value in home currency will be allocated instead of a value determined by a selector master.
NeedFixedDualValue 1 1 boolean If TRUE, specifies that a fixed value in dual currency will be allocated instead of a value determined by a selector master.

AllocationRequirementsList

Contains the requirements in a rule master listed in an allocation chain master.

Child elements

Name Minimum Maximum Type Description
Requirements 0 unbounded AllocationRequirements The requirements for an Allocation run contained in one rule master.

AllocationPrompts

This element contains prompts for those values which have not been specified in the rule master, but are required by the Allocation run.

Child elements

Name Minimum Maximum Type Description
RuleCode 1 1 typeGeneralCode The code of the rule master you want to use.
FixedHomeValue 0 1 typeMoney The fixed value in home currency to be allocated instead of a value determined by a selector master.
FixedDualValue 0 1 typeMoney The fixed value in home currency to be allocated instead of a value determined by a selector master.

AllocationPromptsList

This element contains prompts for those values which have not been specified in any of the rule masters listed in a chain master.

Child elements

Name Minimum Maximum Type Description
PromptData 0 unbounded AllocationPrompts The values which are required by one rule master in the chain.

txnPostData

Child elements

PostTo 1 1 typeCtDocPost
