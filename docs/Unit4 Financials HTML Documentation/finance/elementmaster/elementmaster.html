Root Elements

Element

This element holds an element master.

ElementFinderRequest

This service allows you to search for element masters.

ElementFinderResponse

This is the response that is returned when you search for element masters.

ElementMasterRequest

This service allows you to maintain element masters.

ElementMasterResponse

This is the response that is returned when you maintain element masters.

LastTransactions

Holds details of the last transactions posted against the specified element. These details are read-only and are applicable only to customer/supplier elements and umbrella elements.

Trader

This element holds a trader.

TraderRequest

This service allows you to maintain traders.

TraderResponse

This is the response that is returned when you maintain traders.
