Root Elements

Selection

This element contains the selected data, which is returned from the server.

SelectionWithFlexiFields

The selected data, including flexi-fields, returned to you for browsing.

SelectorRequest

This service allows you to submit selection criteria to retrieve data for browsing.

SelectorResponse

This is the response that is returned when you submit selection criteria.

Selection

This element contains the selected data, which is returned from the server.

Child elements

Name Minimum Maximum Type Description
SelType 1 1 SelectorType Indicates the type of selector: the data class you are using, and the source of the data, retrieved from Books, Intray, Archive or Self.
Header 1 1 SelectionRow Contains vocabulary items (representing columns) for the display of data, which is determined by the display criteria in the presenter master which was specified. Only those vocabulary items included in the presenter are used in the header.
DataSet 1 1 RowList Contains the actual data that has been selected.
RowLimitExceeded 1 1 boolean If TRUE, specifies that the data returned is subject to a limit, since the response to your selection criteria exceeded the allowed number of rows.

SelectionWithFlexiFields

The selected data, including flexi-fields, returned to you for browsing.

Child elements

Name Minimum Maximum Type Description
SelType 1 1 SelectorType Indicates the type of selector: the data class you are using, and the source of the data, retrieved from Books, Intray, Archive or Self.
Header 1 1 HeaderColumnList This element contains a list of header columns.
DataSet 1 1 RowList This element contains a list of rows of data.
RowLimitExceeded 1 1 boolean If set, the maximum number of rows has been exceeded.

SelectDetailsRequest

The SelectDetailsRequest type has Request as its base type.

Child elements

Name Minimum Maximum Type Description
SelType 1 1 DetailsLocation Indicates the source of the data, which may be retrieved from Books, Intray, Archive or Self.
GslCode 1 1 typeGeneralCodeB The code of the selector master you want to use. GslCode is used to look up hints, and also overrides the SelType above.
WholeDocuments 1 1 boolean If TRUE, specifies that you want to browse whole documents for the lines selected.
RowLimit 0 1 typeULong The maximum number of rows you want your search to return.
ApplicationFilter 1 1 VocListData The search criteria to be applied by the application when selecting the data. These are as set up in the selector master.
UserFilter 1 1 VocListData The additional search criteria specified by the user, to be applied when selecting the data. This may be, for example, a document code or a range of codes.
HierarchyFilter 0 1 VocListData The additional search criteria which will be applied to select data within specified hierarchies.
FlexiFieldFilter 1 1 FlexiFieldVocListData The additional search criteria which will be applied to select data from specified flexi-field masters.
Columns 1 1 HeaderColumnList The columns representing vocabulary items that you want the server to return.
