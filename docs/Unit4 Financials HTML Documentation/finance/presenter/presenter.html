Root Elements

Presenter

This element holds a presenter master.

This service allows you to maintain presenter masters.

PresenterResponse

This is the response that is returned when you maintain presenter masters.

Complex Types

UserExit

This element contains details of a user exit routine which can be invoked from Browse details.

Child elements

Name Minimum Maximum Type Description
File 0 1 string The path and filename of the DLL to run from this user exit. If a .DLL file is named here, it MUST NOT be changed by the XMLi programmer, since it is not maintainable in Financials or XMLi. It is maintained only in the Client/server version, and must be resent to the server with the .DLL intact, so that future use of this presenter by a Client/Server user is not adversely affected.
ActionCode 1 1 typeGeneralCode The name of the action code argument passed to a user-defined exit routine in a DLL file. You can leave this blank to pass an empty action code.
Caption 1 1 typeShortName A label for this user exit routine.
UserExtension 0 1 ExtensionRef The code of the user extension which will be available to users when this presenter is used in Browse Details.

VocabListItem

This element contains a vocabulary item used by this presenter master, and its display settings.

Child elements

Name Minimum Maximum Type Description
VocabID 0 1 typeCtOneOfN Specifies the vocabulary item to use in this presenter master.
FFMasterCode 0 1 typeGeneralCode The flexi-field master code.
FFColumnName 0 1 typeFlexiColumnName The flexi-field column name.
FFElmLevel 0 1 typeElmLevel The flexi-field element level.
Protected 1 1 boolean Specifies whether users may change the vocabulary item at run time. When TRUE, the value is protected and cannot be changed.
ColTitle 1 1 typeText Specifies the title to use for this vocabulary item. This defaults to the first 20 characters of the vocabulary name.
DispWidth 1 1 short The number of characters (or width) reserved for displaying the information determined by this vocabulary item.
DispOffset 1 1 short The character position where the displayed information will start. The required number of characters of information, determined by DispWidth, is presented from this position onwards. The offset is measured from the left. This number cannot be greater than the DispWidth value.
Cumulative 1 1 boolean If TRUE, specifies that an accumulated value will be displayed. This is only valid when the VocabID is Document, Dual, Element, or Home value.
ReverseSign 1 1 boolean If TRUE, specifies that the sign on column values is reversed. For example, an original value of 100 is displayed as -100, and an original value of -100 is displayed as 100.
