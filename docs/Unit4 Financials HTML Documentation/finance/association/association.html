Root Elements

asoKey

This element holds generic association key information identifying document lines and statement lines.

asoKeyList

Contains a list of the keys for the lines in the current folder.

Complex Types

asoKey

This element holds generic association key information identifying document lines and statement lines.

Child elements

Name Minimum Maximum Type Description
CmpCode 1 1 typeGeneralCode The code for the company in which the line exists.
Code 1 1 typeGeneralCode The code of the document master or statement master containing the line.
Number 0 1 typeDocNumAlpha The document number or statement number.
LineNumber 1 1 int The line number.
LineType 1 1 typeCtAsoLineType Identifies whether the line is a document line or a statement line.

asoKeyList

Contains a list of the keys for the lines in the current folder.

Child elements

Name Minimum Maximum Type Description
Key 0 unbounded asoKey Contains association key information identifying document lines and statement lines.

asoKeyFolder

Contains key information for the lines in this folder.

Child elements

Name Minimum Maximum Type Description
Keys 0 1 asoKeyList Contains a list of the keys for the lines in this folder.

asoKeyFolders

Contains the lists of debit and credit lines (when performing a MatchingRequest) or the lists of document and statement lines (when performing a BankReconciliationRequest) which the Assistant will attempt to sort into groups.

Child elements

Name Minimum Maximum Type Description
Folder 1 2 asoKeyFolder Contains key information for the lines in this folder.

AssistantScript

This element holds a script for use in Assisted Matching, Background Matching, Assisted Reconciliation or Background Reconciliation.

Child elements

Name Minimum Maximum Type Description
Line 0 unbounded typeScriptText A line in the script.

ProgressStatistics

This element contains statistics of the lines that the Assistant has processed so far.

Child elements

Name Minimum Maximum Type Description
SelectedLinesFolder1 0 1 typeULong The number of lines in Folder 1.
SelectedLinesFolder2 0 1 typeULong The number of lines in Folder 2.
GroupedLinesFolder1 1 1 typeULong The number of lines in Folder 1 that have been placed into groups.
GroupedLinesFolder2 1 1 typeULong The number of lines in Folder 2 that have been placed into groups.
CommittedLinesFolder1 0 1 typeULong The number of lines in Folder 1 that have been committed.
CommittedLinesFolder2 0 1 typeULong The number of lines in Folder 2 that have been committed.
ComparisonsMade 1 1 typeULong The total number of comparisons that the Assistant has made.
GroupsCreated 1 1 typeULong The number of groups that the Assistant has produced.
GroupsCommitted 0 1 typeULong The number of groups that the Assistant has committed.
Phase 0 1 typeCtBrrPhase The stage that the assisted or background process has reached.
