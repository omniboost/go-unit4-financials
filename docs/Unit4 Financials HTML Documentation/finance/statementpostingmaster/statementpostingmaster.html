http://www.coda.com/efinance/schemas/statementpostingmaster

Root Elements

StatementPostingMaster

This element holds a statement posting master.

StatementPostingMasterRequest

This service allows you to maintain statement posting masters.

StatementPostingMasterResponse

This is the response that is returned when you maintain statement posting masters.

Complex Types

FolderList

Contains the list of user-defined folders.

Child elements

Name Minimum Maximum Type Description
Folder 0 unbounded FolderItem A user-defined folder on the list.

FolderItem

A user-defined folder on the list.

Child elements

Name Minimum Maximum Type Description
FolderName 1 1 typeShortName The name of a user-defined folder.
StatementSelectorCode 1 1 typeGeneralCode The code of the selector master that you want to use to retrieve the statement lines for this folder.
StatementPresenterCode 1 1 typeGeneralCode The code of the presenter master that you want to use to display the statement lines in this folder.
DocumentPresenterCode 1 1 typeGeneralCode The code of the presenter master that you want to use to display the proposed documents before they are created and posted.
DocumentDateSource 1 1 typeCtDocumentDate How the document date is derived: Statement Date or From file.
OnPostingErrorAction 1 1 typeCtOnPostingError The action you want the system to take, if an error occurs during posting: Attempt to post to the Intray or Abort the posting.
ReconcileToBooks 1 1 boolean If TRUE, specifies that, if the transaction type is set to Reconcile, the system reconciles the transaction after posting it to the Books.
ReconcileToIntray 1 1 boolean If TRUE, specifies that, if the transaction type is set to Reconcile, the system reconciles the transaction after posting it to the Intray.

StatementPostingMaster

This element holds a statement posting master.

Child elements

Name Minimum Maximum Type Description
CmpCode 1 1 typeGeneralCode The code of the company in which the statement posting master exists.
Code 1 1 typeGeneralCode The code for this statement posting master.
TimeStamp 0 1 short The TimeStamp value for the statement posting master in the database.
Name 1 1 typeName The full name of the master.
ShortName 0 1 typeShortName The abbreviated name of the master.
StatementCode 0 1 typeGeneralCode The code of the default statement type that you want to use with this master.
StatementCurrencyCode 0 1 typeGeneralCode The currency code of the statements that you want to post using this master.
ReverseStatementValue 1 1 boolean If TRUE, specifies that at run time the statement and balance values are displayed with reversed positive/negative signs.
PostedItemsPresenterCode 0 1 typeGeneralCode The code of the presenter master that you want to use to display the statement lines in the Posted Items folder.
RemainingItemsPresenterCode 0 1 typeGeneralCode The code of the presenter master that you want to use to display the statement lines in the Remaining Items folder.
PostedItemsFolderTitle 0 1 typeText The name of the folder in which posted lines are displayed at run time.
RemainingItemsFolderTitle 0 1 typeText The name of the folder in which any lines not assigned to other folders are displayed at run time.
Folders 0 1 FolderList Contains the list of user-defined folders.
