Root Elements

ProgressStatistics

This element contains the statistics for the progress made with the tax transfer process currently running.

This element contains the required parameters from running the tax transfer process.

TaxTransferRequest

This service allows you to transfer tax.

Complex Types

StartParams

This element contains the required parameters from running the tax transfer process.

Child elements

Name Minimum Maximum Type Description
CmpCode 1 1 typeGeneralCode The code for the company in which the tax transfer process is started.
TaxTransferMasterCode 1 1 typeGeneralCode The code for the tax transfer master used to run Tax Transfer.
DeferredTaxFromDate 0 1 typeDate When transferring deferred VAT the process is initiated by a date range. All lines whose paid date or undo-match date is later than this date, and earlier than the date set in the DeferredTaxToDate element will be selected for processing.
DeferredTaxToDate 0 1 typeDate When transferring deferred VAT the process is initiated by a date range. All lines whose paid date or undo-match date is earlier than this date, and later than the date set in the DeferredTaxFromDate element will be selected for processing.
SelCode 0 1 typeGeneralCode The code for the selector master which the tax transfer process uses.
SelectorVocabList 0 1 VocListData The list of vocabulary items defined in the selector master.
ReportOnly 1 1 boolean Specifies that this is a 'report only' run. Meaning that you can see what values would be transferred if you ran a full transfer process, but the database is not updated.
TransferDocDate 0 1 typeDate The date of the generated transfer documents.
TransferDocPeriod 0 1 typePeriod The year and period of the generated transfer documents.
Description 0 1 typeDescr The default description for the generated transfer document lines.
LogTitle 1 1 typeLogTitle The title of the log, entered by the user who ran the transfer process.
LogLevel 1 1 typeCtLogLevel The amount of information you want to display in the log. You can set this element to Summary, Detailed or Highly detailed.

ProgressStatistics

This element contains the statistics for the progress made with the tax transfer process currently running.

Child elements

Name Minimum Maximum Type Description
LinesToProcess 0 1 typeULong The total number of customer/supplier lines to be processed.
LinesProcessed 1 1 typeULong The number of customer/supplier lines processed so far.
LinesRejected 1 1 typeULong The number of lines which have been processed and rejected.
Sorting 1 1 boolean Sorts the selected documents into a transfer summary order. When this process is running, the Sorting element is true.
TransferSummariesToProcess 0 1 typeULong The total number of transfer documents to be processed.
TransferSummariesProcessed 1 1 typeULong The number of transfer documents processed so far.
TransferSummariesRejected 1 1 typeULong The number of transfer documents which have been processed and rejected.
