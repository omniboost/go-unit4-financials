UserExtensionResults

Contains information about the user extensions: whether they were successful, and the values they returned. If only the source side is present for this statement line, then all the contra elements should be omitted. Similarly, if only the contra side is present for this statement line, then all the source elements should be omitted. Error element, if present, indicates that there was an instruction to the User Extension (either to validate or lookup an account) that could not be satisfied due to a problem with the User Extension (either it has not been set up yet, or cannot be located).
