Root Elements

ReminderMasterRequest

This service allows you to maintain reminder masters.

ReminderMasterResponse

This is the response that is returned when you maintain reminder masters.

Complex Types

LetterSetupItem

This element holds the details for one reminder letter.

Child elements

Name Minimum Maximum Type Description
Severity 1 1 short The severity level of the reminder letter.
DateSetting 1 1 short The number of days that must elapse after the due or document date, or since the last reminder letter, before a letter of this severity can be sent.
DateType 1 1 typeCtDateType The length of time of a date unit. This is always Days.
LetterTemplate 1 1 typeLetterTemplateB A free-format text field containing the name of the word processing template used to produce the reminder letter.
Include 1 1 boolean Specifies that all detail lines that meet the selection criteria for the specified element and currency must be included in the reminder letter.
MinimumTrigger 1 1 decimal The minimum value, in home currency, that must be outstanding to generate a reminder letter at this level; that is, the sum of all detail lines that meet the requirements.
MinimumTotal 1 1 decimal The minimum total value, in home currency, that must be outstanding to generate a reminder letter at this level; that is, the sum of all detail lines that meet the requirements, plus any outstanding from letters of a lower level (applies only if Include is TRUE).
DiaryEntry 1 1 boolean Specifies whether the letter creates an entry in the Diary.
Priority 0 1 typeDiaryPriority The priority level of the task. This can be High, Medium or Low.
TaskRequired 0 1 typeDescr Whatever action is needed as a result of sending the reminder letter.
ActionCode 0 1 typeGeneralCode The required action code, as defined in the relevant entity master for the current company. Action codes refer to the type of action which you take when there is a payment dispute with a customer.
ActionDate 0 1 typeHSDate The date that the action is due. A soft date will be interpreted during the confirmation stage of the Reminder Letters process.

Child elements

ReminderMaster

This element holds a reminder master.

Child elements

Name Minimum Maximum Type Description
CmpCode 1 1 typeGeneralCode The code for the company in which the reminder master is being maintained.
Code 1 1 typeGeneralCode The code for this reminder master.
TimeStamp 1 1 short The TimeStamp value for this reminder master in the database.
Name 1 1 typeName The full name for this reminder master.
ShortName 1 1 typeShortNameB The abbreviated name for this master.
ElementLevel 1 1 typeElmLevel The level of the element that holds the address to which the reminder letter is sent.
Currency 1 1 typeCtExtractCurrency The currency used on a reminder letter.
BaseDate 1 1 typeCtBaseOnDate The date used to calculate when a reminder letter is triggered.
SelectorCode 1 1 typeGeneralCode The code of the selector master to use to select the document lines.
PresenterCode 1 1 typeGeneralCode The code of the presenter master that will be used to display the exploded view of reminder letter details.
LetterSetupList 0 1 LetterSetupList This element holds the reminder letter details for one reminder master.
