BalanceService

Namespace: http://www.coda.com/efinance/schemas/balance/balance-3.0/webservice
WSDL: finance-balance-balance.wsdl
Path: services/finance/balance/balance-3.0

The following methods are available on this endpoint:

  • Add
  • Amend
  • Get
  • Save
  • Update

Add

Input Parameters

AddRequest
Name Minimum Maximum Type Description
Balance 1 1 balance Contains the information for the balance you want to add.

Return Value

AddResponse
Name Minimum Maximum Type Description
Key 1 1 balanceKey The balance key for the balance you attempted to create.

Amend

Input Parameters

AmendRequest
Name Minimum Maximum Type Description
Balance 1 1 balance Contains the amended balance information.

Return Value

AmendResponse
Name Minimum Maximum Type Description
Key 1 1 balanceKey The balance key for the balance you attempted to amend.
Added 0 1 boolean If TRUE, indicates that the amended balance information has been added.

Get

Input Parameters

GetRequest
Name Minimum Maximum Type Description
Key 1 1 balanceKey Contains the key identifying the balance information you want to retrieve.

Return Value

GetResponse
Name Minimum Maximum Type Description
Key 1 1 balanceKey Contains the key identifying the balance information you wanted to retrieve.
Balance 1 1 balance The balance information you have retrieved.

Save

Input Parameters

SaveRequest
Name Minimum Maximum Type Description
Balance 1 1 balance Contains the information for the balance you want to save.

Return Value

SaveResponse
Name Minimum Maximum Type Description
Key 1 1 balanceKey The balance key for the balance you attempted to save.
