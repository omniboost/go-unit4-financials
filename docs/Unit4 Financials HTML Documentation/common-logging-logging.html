LoggingService

Namespace: http://www.coda.com/common/schemas/logging/logging-12.0/webservice
WSDL: common-logging-logging.wsdl
Path: services/common/logging/logging-12.0

The following methods are available on this endpoint:

  • AppendLines
  • Create
  • Delete
  • Get
  • GetFilteredLog
  • GetFilteredLogEx
  • List

AppendLines

Input Parameters

AppendLinesRequest
Name Minimum Maximum Type Description
Key 1 1 LogKey The key that identifies a log to which log lines should be appended.
Lines 1 1 LineDetailData The log lines to be appended to this log.

Return Value

AppendLinesResponse
Name Minimum Maximum Type Description
Key 1 1 LogKey The key that identifies this log.
Lines 1 1 LineDetailData The log lines that were appended.

Name Minimum Maximum Type Description
