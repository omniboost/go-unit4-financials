Input Parameters

UpdateWithPositionsRequest
Name Minimum Maximum Type Description
User 1 1 User Contains the information for the Coda user master you want to update.

This allows you to make any changes and attempt another update.
UserCapabilityList 1 1 UserCapabilityList The list of capabilities by company or group.

ExternallyAddedPositions 0 1 Positions The list of positions with which the user has been associated by someone else while you were editing the user master.
ExternallyRemovedPositions 0 1 Positions The list of positions from which the user has been removed by someone else while you were editing the user master.
