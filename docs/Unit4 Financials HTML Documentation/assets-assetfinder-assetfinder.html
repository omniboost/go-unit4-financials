AssetFinderService

Namespace: http://www.coda.com/eassets/schemas/assetfinder/assetfinder-12.0/webservice
WSDL: assets-assetfinder-assetfinder.wsdl
Path: services/assets/assetfinder/assetfinder-12.0

The following methods are available on this endpoint:

  • List
  • ListChildren
  • ListIntray
  • Search

List

Input Parameters

ListRequest
Name Minimum Maximum Type Description
Filter 1 1 AssetReqKey A filter that selects the assets to be listed.

Return Value

ListResponse
Name Minimum Maximum Type Description
Keys 1 1 AssetKeyData Contains a list of keys identifying the assets that have been returned from the asset finder.

ListChildren

Input Parameters

ListChildrenRequest
Name Minimum Maximum Type Description
Keys 1 1 AssetKeyData This element contains key information for the list of child assets.

Return Value

ListChildrenResponse
Name Minimum Maximum Type Description
Keys 1 1 AssetKeyData Contains a list of keys identifying the child assets that have been returned from the asset finder.

Input Parameters

ListIntrayRequest
Name Minimum Maximum Type Description

Return Value

ListIntrayResponse
Name Minimum Maximum Type Description

Search

Input Parameters

SearchRequest
Name Minimum Maximum Type Description
Filter 1 1 SearchFilter The search criteria.
Intray 1 1 boolean True to search the Assets Intray, false to ignore the Assets Intray.

Return Value

SearchResponse
Name Minimum Maximum Type Description
Filter 1 1 SearchFilter The search criteria.
Keys 0 1 AssetSearchDataList The search results.
ListIsComplete 1 1 boolean If true, the records returned are the complete results of the search. If false, the results were truncated.
