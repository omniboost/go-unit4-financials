ExtensionMasterService

Namespace: http://www.coda.com/common/schemas/extensionmaster/extensionmaster-2.0/webservice
WSDL: common-extensionmaster-extensionmaster.wsdl
Path: services/common/extensionmaster/extensionmaster-2.0

The following methods are available on this endpoint:

  • Add
  • Delete
  • Get
  • List
  • Update

Add

Input Parameters

AddRequest
Name Minimum Maximum Type Description
Extension 1 1 Extension Contains the information for the extension master you want to create.

Return Value

AddResponse
Name Minimum Maximum Type Description
Code 1 1 typeGeneralCode The code for the extension master that you created or attempted to create.
TimeStamp 1 1 typeWord The TimeStamp value for this extension master in the database.

Delete

Input Parameters

DeleteRequest
Name Minimum Maximum Type Description
Key 1 1 GlobalKey Contains the key identifying the element master you want to delete.

Return Value

DeleteResponse
Name Minimum Maximum Type Description
Code 1 1 typeGeneralCode The code for the extension master that you deleted or attempted to delete.

Get

Input Parameters

GetRequest
Name Minimum Maximum Type Description
Key 1 1 GlobalKey Contains the key identifying the extension master you want to retrieve from the database.

Return Value

GetResponse
Name Minimum Maximum Type Description
Code 1 1 typeGeneralCode The code for the extension master that you retrieved or attempted to retrieve.
Extension 0 1 Extension Contains the extension master you have retrieved from the database.

Input Parameters

ListRequest
Name Minimum Maximum Type Description

Return Value

ListResponse
Name Minimum Maximum Type Description
