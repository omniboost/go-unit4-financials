PIMMatchingSelectorService

Namespace: http://www.coda.com/pim/schemas/matching/pimmatchingselector-13.0/webservice
WSDL: pim-matching-pimmatchingselector.wsdl
Path: services/pim/matching/pimmatchingselector-13.0

The following methods are available on this endpoint:

  • ExplodeMatch
  • RunGroupingAssistant
  • Select
  • SelectFolders
  • SelectMatchedInvoicesToUndo
  • SelectWIPInvoices
  • SingleInvoiceSelect
  • SingleInvoiceSelectWIPInvoices

ExplodeMatch

Input Parameters

ExplodeMatchRequest
Name Minimum Maximum Type Description
Invoice 1 1 InvoiceKey Contains the key identifying the matched invoice that you want to explode.
ExplodeInvoicePresenterMaster 1 1 Presenter The code for the presenter master you want to use to display the exploded invoice details.
ExplodeLRNPresenterMaster 0 1 Presenter The code for the presenter master you want to use to display the exploded LRN details

Return Value

ExplodeMatchResponse
Name Minimum Maximum Type Description
InvoiceSelection 1 1 Selection The purchase invoices in the exploded match.
LRNSelection 1 1 Selection The LRNs in the exploded match.
MatchReference 1 1 typeULong The match reference.

RunGroupingAssistant

Input Parameters

RunGroupingAssistantRequest
Name Minimum Maximum Type Description
MatchingMaster 1 1 MatchingMaster The code of the matching master you want to use.
Invoices 1 1 Selection The purchase invoice documents available for grouping.
LRNLines 1 1 Selection The LRN lines available for grouping.

Return Value

RunGroupingAssistantResponse
Name Minimum Maximum Type Description
Group 0 unbounded Group This element contains a group created by the grouping assistant.
GroupedInvoice 0 unbounded InvoiceKey Identifies the selected purchase invoice documents that have been grouped.
UngroupedInvoice 0 unbounded InvoiceKey Identifies any selected purchase invoices that have not been grouped.
GroupedLRNLine 0 unbounded LRNLineKey Identifies the selected LRN lines that have been grouped.
UngroupedLRNLine 0 unbounded LRNLineKey Identifies any selected LRN lines that have not been grouped.
