Glossary

API
Application Programming Interface - a program-level interface that allows user-written code to interact with the system. For Web Services this is XMLi.
ESI
Enterprise Servlet Infrastructure - the CODA technology used to provide the browser-hosted front-end.
SOAP
Simple Object Access Protocol - a simple XML-based protocol for information exchange. Note that literal encoding will be used (as opposed to RPC-style encoding).
Stack
The set of protocols that allow Web services to be described and supported.
UDDI
Universal Description, Discovery and Integration - this is a directory of available Web services, usually organised by industry sector. The directory identifies the host, the service and the parameters.
Web Service
Any piece of software that makes itself available across the internet using standardised XML messaging.
WSDL
Web Service Description Language - describes the Web service and its parameters and returns.
XML element
A data item in an XML message (distinguished from a CODA elementmaster).
XMLi
XML for Interfacing - CODA's XML interface.
