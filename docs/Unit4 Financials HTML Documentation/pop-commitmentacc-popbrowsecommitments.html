POPBrowseCommitmentsService

WSDL: pop-commitmentacc-popbrowsecommitments.wsdl
Path: services/pop/commitmentacc/popbrowsecommitments-1.0

The following methods are available on this endpoint:

  • BrowseCommitmentDetails
  • BrowseCommitments

BrowseCommitmentDetails

Input Parameters

BrowseCommitmentDetailsRequest
Name Minimum Maximum Type Description
SelectionType 1 1 SelectorType Specifies the source of the data from which the selection is made.
Filter 1 1 VocListData The criteria to be applied when selecting the data.
Columns 1 1 CellList The columns that will hold the selected data.
DocType 1 1 string The type of commitment to which the selected data relates.
Document 1 1 DocumentKey The commitments document key (company code, document code, and document number).
PeriodEnd 1 1 typePeriodB The period end to be used for period-end commitments.
ViewName 1 1 string The name of the dataview used by the selector.

Return Value

BrowseCommitmentDetailsResponse
Name Minimum Maximum Type Description
Selection 1 1 PopSelection The selected data returned to you for browsing.
