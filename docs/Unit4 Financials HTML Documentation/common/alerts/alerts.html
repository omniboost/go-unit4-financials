Alert

This element holds alert details.

A translatable message with parameters that can be sent to a number of users.

Child elements

Name Minimum Maximum Type Description
Id 1 1 typeDbGuid

A unique identifier for the alert.
Priority 1 1 typeComAlertPriority

The priority of the alert.
EmailSubject 0 1 typeLongDescriptionb

The email subject used when issuing email alerts.
Translate 1 1 typeBoolean

Specifies whether the alert text is translated.

If this is null it means that the summary and detail are plain text with the parameter insert as per reason messages.

Summary 1 1 typeText

The summary text for an alert or a reference to the translated summary text.
Detail 1 1 typeText

The detailed text for an alert or a reference to the translated detailed text.
Uri 0 1 typeUri

The URI part of the data type name.
SummaryParameterList 1 1 AlertSummaryParameterList

A list of summary parameters for the alert.
DetailParameterList 1 1 AlertDetailParameterList

A list of detailed parameters for the alert.
Transport 0 1 MessageTransportKey

The message transport used to transmit the alert.
