http://www.coda.com/common/schemas/functionalsecurity

Root Elements

FunctionalSecurityRequest

This document enables you to maintain functional security through the Administration Console server.

FunctionalSecurityResponse

This is the response document that is returned when you maintain functional security through the Administration Console server.

Complex Types

FunctionalSecurityUnion

This element holds functional security parameters.

Child elements

Name Minimum Maximum Type Description
Hash 1 1 string A value which identifies a combination of the user capability and associated role timestamp values.
CapabilityCode 1 1 typeGeneralCode The code for the capability master.
CapabilityTimeStamp 1 1 short The timestamp value for the capability master.
Roles 1 1 RolesList This element holds a list of role masters.
FunctionalSecurityList 1 1 FunctionalSecurityList This element holds the functional security list.

RolesList

Child elements

Child elements
