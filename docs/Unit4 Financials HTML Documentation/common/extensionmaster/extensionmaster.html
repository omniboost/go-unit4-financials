http://www.coda.com/common/schemas/extensionmaster

Root Elements

ExtensionMasterRequest

This service allows you to maintain extension masters.

ExtensionMasterResponse

This is the response that is returned when you maintain extension masters.

Complex Types

ExtensionKey

This element is a key for requesting a number of extension masters from the database.

Child elements

Name Minimum Maximum Type Description
MaxKeys 1 1 int The maximum number of extension masters to be selected. A value of zero indicates no limit.
Code 1 1 typeGeneralCode The code to use for selecting extension masters.
ShortName 0 1 typeShortNameB The short name to use for selecting extension masters.
Usage 1 1 typeGenericOon The application area in which the extension masters are used.

Extension

This element holds an extension master.

Child elements

Name Minimum Maximum Type Description
Code 1 1 typeGeneralCode The code for this extension master.
Name 1 1 typeName The full name for this extension master.
ShortName 1 1 typeShortNameB The abbreviated name for this extension master.
TimeStamp 1 1 short The TimeStamp value for this extension master in the database.
Usage 1 1 typeGenericOon The application area in which this extension master is used.
Type 1 1 typeComExt The type of this extension master. Currently this must be Java Class.
Location 0 1 string The fully qualified name of the Java class.
Enable 1 1 boolean Indicates whether this extension master is enabled for use.
Script 0 1 string The code that makes up the script.
