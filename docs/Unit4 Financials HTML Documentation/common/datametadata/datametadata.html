http://www.coda.com/common/schemas/datametadata

Root Elements

DataMetadataRequest

This service allows you to maintain the following types of metadata masters: data source, metadata set, metadata object, and enumeration. It also provides some utility requests.

DataMetadataResponse

This is the response that is returned when you maintain metadata masters or use the utility requests.

Simple Types

typeDataMetadataCategoryType

This enumeration encodes a classification of attributes according to whether they can be grouped or summed.

Base type typeBaseEnum
Enumeration Values
data_metadata_grouped
Attribute values can be grouped.
data_metadata_summed
Attribute values can be summed.
data_metadata_unique
Attribute values cannot be either grouped or summed.

typeDataMetadataChecksumType

This enumeration encodes the options for performing checksum validation.

Base type typeBaseEnum
Enumeration Values
data_metadata_checksum_none
There is no checksum validation.
data_metadata_checksum_column
Individual checksum values are calculated for each applicable column.
data_metadata_checksum_row
A single checksum value is calculated for all applicable columns on a row.

typeDataMetadataDescription

The type of descriptions for metadata items.

Base type string
Minimum length 1
Maximum length 40

typeDataMetadataId

A system-generated globally unique identifier.

Base type decimal
Minimum value -9999999999999999999999999999
Maximum value 9999999999999999999999999999

This service allows you to maintain the following types of metadata masters: data source, metadata set, metadata object, and enumeration. It also provides some utility requests.
