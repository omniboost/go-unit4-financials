http://www.coda.com/common/schemas/usermaster

Root Elements

UserMasterRequest

This service allows you to maintain Coda user masters via the Administration Console server.

UserMasterResponse

This is the response that is returned when you maintain user masters via the Administration Console server.

Simple Types

UserMasterElmLevel

The element level of a default element specified on the user master.

Base type short
Minimum value 0
Maximum value 8
