http://www.coda.com/common/schemas/homepage

Root Elements

HomePageRequest

This service allows you to maintain the Home Page of the current user.

HomePageResponse

This is the response that is returned when you maintain Homepages.

Content

This element holds the content available in a frame on the Home Page.

Attributes

Name Type Use Default Description
id ID optional An identifying value that links this content with a frame.
lastupdate dateTime optional The date and time when this content was last updated.
pending boolean optional When TRUE, the content is currently being requested or is waiting to be requested.
refresh boolean optional Indicates whether the user is permitted to refresh this content.
timestamp int optional The TimeStamp value of the content provider master used to retrieve this content.
type typeContentType optional The type of content. Currently only XML is supported.

Child elements

Name Minimum Maximum Type Description
Exception 0 1 Reason Contains an error message if a request for this content fails.
1 1

Name Minimum Maximum Type Description

ContentProviderList

This element contains details of the content provider masters available to this Home Page.

Child elements

Name Minimum Maximum Type Description
ContentProvider 0 unbounded ContentProviderListItem This element contains details of a content provider master available to this Home Page.

ContentProviderListItem

This element contains details of a content provider master available to this Home Page.

Attributes

Name Type Use Default Description
id ID optional An identifying value that links this content provider master with a frame.

Child elements

Name Minimum Maximum Type Description
Code 1 1 typeGeneralCode The code for the content provider master.
Name 1 1 typeName The full name for the content provider master.
ShortName 1 1 typeShortNameB The abbreviated name for the content provider master.
ExtensionRef 1 1 ExtensionRef Identifies the system extension used by the content provider master.
ExtensionTimeStamp 1 1 short The TimeStamp value for this system extension in the database.
AllowManualRefresh 1 1 boolean Indicates whether the user is permitted to refresh the data displayed by the content provider master.
ModifiableConfiguration 1 1 ProviderModifiableConfiguration Contains a configurable setting which the user is permitted to modify in this Home Page.
Configuration 1 1 ProviderConfiguration Contains the settings which may be configured for the content provider master.
AddToActiveDesktop 1 1 boolean Indicates whether this content provider master can be displayed on the user's Active Desktop.
