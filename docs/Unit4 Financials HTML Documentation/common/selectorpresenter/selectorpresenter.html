http://www.coda.com/common/schemas/selectorpresenter

Root Elements

SelectorPresenterRequest

This service allows you to import and export metadata selectors and presenters.

SelectorPresenterResponse

This is the response that is returned after issuing a request to export or import selectors and presenters.

Complex Types

ExportRequest

The ExportRequest type has Request as its base type.

Child elements

Name Minimum Maximum Type Description
Selectors 1 1 KeyList A list of selectors.
Presenters 1 1 KeyList A list of presenters.

ExportRequestVerb

The ExportRequestVerb type has RequestVerb as its base type.

Exports a specified list of selectors and presenters and any supporting data objects to an XML file.

Child elements

Name Minimum Maximum Type Description
Request 1 unbounded ExportRequest

ExportResponse

The ExportResponse type has Response as its base type.

Child elements

Name Minimum Maximum Type Description
SelectorPresenterExport 1 1 SelectorPresenterExport This element holds details of exported metadata.
SelectorErrors 1 1 ExportErrorList This element contains a list of errors which occurred while exporting selectors.
PresenterErrors 1 1 ExportErrorList This element contains a list of errors which occurred while exporting presenters.
ExportSuccess 1 1 boolean If set, indicates that the export process was successful.

ExportResponseVerb

The ExportResponseVerb type has ResponseVerb as its base type.

Contains the responses to your requests with this verb.
