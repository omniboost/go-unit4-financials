ActionType

Allows for the Application to render action buttons using different icons and/or color schemes to indicate the action's behaviour. For instance the "Approve" action button might have a green background whereas the "Deny" would be displayed in red. Use "Neutral" for actions where the action type cannot be determined.

Base type string
Enumeration Values
Negative
Suitable for actions like "Deny", "Decline" or "Cancel".
Neutral
Suitable for actions like "Forward", or "Request information".
Positive
Suitable for actions like "Approve", or "Allow".
