UserInterfaceSnapshot

This element holds settings relating to a user's view, for example, a bookmarked view of Aged Analysis.

Child elements

Name Minimum Maximum Type Description
Usercode 1 1 typeCode

The code of the user to whom the settings apply. For example, the user who has bookmarked the view.

Area 1 1 typeDescription

The area of the product where the settings apply. For example, Aged Analysis.

Specifics 1 1 typeTextAnyDb255

Key values relating to the specific area. For example, company code and ledger code for Aged Analysis.

Sequence 1 1 typeLong

A sequence number for when data has to be split into several rows.

Data 1 1 typeTextAnyDbQuotes

A JSON formatted string containing the setting values.

AccessTime 0 1 typeHardTime

A timestamp with the last time these settings were accessed (read or write).
