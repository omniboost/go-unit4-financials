The code of the user responsible for this task.

The company where the task is set up.

The Type of the log generated by this task (if any).

A unique identifier for the log.

The date and time when the task was started.

The status of the task.

RichStatus 0 1 typeTranslatable

A description of the task status.

UnitsComplete 0 1 typeULongB

The number of units already completed by the task.

UnitsTotal 0 1 typeULongB

The total number of units in this task.

PercentageComplete 1 1 typePercentageb

The percentage of the task which has been completed.

ControlRequest 0 1 typeULongB

The control request that is currently being sent to the task. The task should be expected to process these requests sequentially. If the request is the null request, then a request may be set, otherwise the task is deemed to be busy processing the request already set. The task must be left to complete this request processing before another request can be sent to it.

LastTouched 1 1 typeDateTimeb

Indicates the date and time on which the task last modified its status.

SavedState 0 1 SavedTaskState

Saved task state details.

TimeStamp 1 1 short The TimeStamp number of this Task.
