http://www.coda.com/common/schemas/workflowselection

Root Elements

WorkflowSelectionRequest

This service allows you to submit selection criteria to retrieve workflow data.

WorkflowSelectionResponse

This is the response that is returned when you issue a workflow selection request.

Complex Types

WorkflowIds

A list of workflow instance IDs.

Child elements

Name Minimum Maximum Type Description
Id 0 unbounded typeDbGuid A workflow instance ID.

SelectRequest

The SelectRequest type has Request as its base type.

Child elements

Name Minimum Maximum Type Description
SelType 1 1 SelectorType Specifies the source of the data from which the selection is made. The data source is determined by the combination of data class and data location.
GslCode 1 1 typeGeneralCodeB The code of the selector master you want to use. GslCode is used to look up hints, and also overrides the SelType above
RowLimit 0 1 typeULong The maximum number of rows you want your search to return.
ApplicationFilters 0 1 VocListDataList The search criteria to be applied by the application when selecting the data. These are as set up in the selector master.
UserFilter 1 1 VocListData The additional search criteria specified by the user, to be applied when selecting the data.
Columns 1 1 CellList The columns representing vocabulary items that you want the server to return.
