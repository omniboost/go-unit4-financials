ContentProvider

This element holds a content provider master.

Child elements

Name Minimum Maximum Type Description
Code 1 1 typeGeneralCode The code for this content provider master.
Name 1 1 typeName The full name for this content provider master.
ShortName 1 1 typeShortNameB The abbreviated name for this content provider master.
TimeStamp 1 1 short The TimeStamp value for this content provider master in the database.
ExtensionRef 1 1 ExtensionRef Identifies the system extension used by this content provider master.
Configuration 1 1 ProviderConfiguration Contains the settings which may be configured for this content provider master.
ModifiableConfiguration 1 1 ProviderModifiableConfiguration Contains the name of a configurable setting which users are permitted to modify via their own Home Page.
AllowManualRefresh 1 1 boolean Indicates whether users are permitted to refresh the data displayed by this content provider master.
RefreshUnit 1 1 short The numerical part of the frequency with which the data displayed by this content provider master is refreshed. For example, if the refresh frequency is 5 hours, this value is 5.
RefreshType 1 1 typeProviderRefreshType The time period part of the frequency with which the data displayed by this content provider master is refreshed. For example, if the refresh frequency is 5 hours, this value is Hours.
AddToActiveDesktop 1 1 boolean Specifies whether this content provider can be displayed on the user's Active Desktop.
