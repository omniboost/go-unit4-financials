Complex Types

GenericBrowseMaster

This element holds a generic browse report master.

Child elements

Name Minimum Maximum Type Description
Code 1 1 typeCode

The code of the generic browse report master.

Name 1 1 typeName

The full name for the master.

ShortName 0 1 typeShortNameB

The abbreviated name for the master.

PrintHeaderAndFooter 1 1 typeComGenericbrowseHfLoc

Specifies when the header and footer are printed.

RowsPerPage 1 1 typeRowsPerPrintPage

The number of data rows per page when the report is printed.

Product 1 1 typeApplicationName

Specifies the product to which the data belongs or that the data is external to the system.

BrowseSelector 1 1 typeCodeB

The code for the selector master used to select the data.

BrowsePresenter 1 1 typeCodeB

The code for the presenter master used to present the data.

Dataview 1 1 typeCodeB

The code for the dataview describing the data. This is calculated from the dataview used by the selector. (The presenter should use the same dataview too).

DataSecurityMaster 0 1 typeCodeB

The code for the data security master that controls the data security that is applied to this master.

DataSecurityMasterProt 1 1 typeBoolean

Specifies whether the data security master on this generic browse report master is protected from being edited when the user runs Generic Browse.

OutputDevice 0 1 typeCodeB

The output device to use to print or transmit rows selected by this generic browse report master.

PrintFormat 0 1 typeCodeB

The print format to use to format rows selected by this generic browse report master.

EmailTemplate 0 1 typeCodeB

The code of an e-mail template master used to provide the subject, body and signature of generated emails.

CommentText 0 1 typeTextB

The text content of the comment.

CommentLocation 0 1 typeComGenericbrowseCommentLocation

The location where the comment appears on the report.

CommentExport 0 1 typeBoolean

Specifies whether the comment is included when the report is exported.

HeaderList 1 1 ColumnDefinitionList

This element holds the details of the header columns.
HeaderEqualWidth 1 1 typeBoolean

Specifies whether header columns are of equal width.
FooterList 1 1 ColumnDefinitionList

This element holds the details of the footer columns.
FooterEqualWidth 1 1 typeBoolean

Specifies whether footer columns are of equal width.
TimeStamp 1 1 short The TimeStamp number of this GenericBrowseMaster.
AddDate 0 1 dateTime The date this GenericBrowseMaster was created.
ModDate 0 1 dateTime The last date this GenericBrowseMaster was modified.
UsrName 0 1 string The code of the user who last modified this GenericBrowseMaster record.
