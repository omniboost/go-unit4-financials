Complex Types

OutputDeviceMaster

This element holds an output device master.

Child elements

Name Minimum Maximum Type Description
Code 1 1 typeCode

The code for this output device master.

Name 1 1 typeName

The name of the output device master.

ShortName 0 1 typeShortNameB

The short name for the output device master.

DeviceType 1 1 typeComDevice

Specifies the type of the device.

PhysicalLocation 0 1 typePhysicalLocation

The physical location of the device (applies only to printers).

DevicePath 0 1 typeLocationb

The physical location of the device (if a printer), or the default e-mail address (if e-mail), or the full path for the filename (if a file). If the device type is FTP, this is the full path for the directory on the remote host to which you want to copy the file being transferred.

Format 1 1 typeComDeviceFormat

The file formats that the device accepts.

Transport 0 1 MessageTransportKey

The key which identifies the message transport master to use.

Host 0 1 typeText

The name of the remote host to which files will be transferred. Applies only to the FTP device type.

Port 0 1 typePortNoB

The port number to use when connecting to the remote host. Applies only to the FTP device type.

Username 0 1 typeTextWildFixed

The user name to use when logging on to the remote host. Applies only to the FTP device type.

Password 0 1 typeTextAnyDbPassword

The password for this user. Applies only to the FTP device type.

Filename 0 1 typeTextWildFixed

The default value for the name on the remote host of the file being transferred. Applies only to the FTP device type.

FtpBinaryData 0 1 typeBoolean

Specifies whether the file being transferred is a binary file. Applies only to the FTP device type.

SendAttachments 0 1 typeBoolean

If true, specifies that, in general, when you output documents as e-mails, file attachments to the documents are sent as attachments to the e-mails. If false, such file attachments are not sent with the e-mails. (Reference attachments are never sent with e-mails.) See the topic 'Sending attachments with e-mails' in the Administration Console online Help for more detail.

RepositoryFolder 0 1 RepositoryFolderKey

A key which identifies a repository folder.

SenderEmailAddress 0 1 typeEmailAddressb

An optional email address to use as the sender.

CopyEmailToSender 0 1 typeBooleanb

If true, a copy of the email is sent to the sender.

TimeStamp 1 1 short The TimeStamp number of this OutputDeviceMaster.
