http://www.coda.com/common/schemas/securityreports

Root Elements

SecurityReportsRequest

This service allows you to generate security reports.

SecurityReportsResponse

This is the response that is returned when you generate security reports.

Simple Types

SecRptCapabilityLevel

A capability level from 1 to 7 (1 is lowest).

Base type int
Minimum value 1
Maximum value 8

SecurityReportOrientation

This element contains report orientation details.

Base type typeBaseEnum
Enumeration Values
by_user
For example, a roles by user report.
by_entity
For example, a users by role report.
