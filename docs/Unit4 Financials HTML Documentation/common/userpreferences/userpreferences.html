http://www.coda.com/common/schemas/userpreferences

Root Elements

UserPreferencesRequest

This service allows you to maintain Favourites lists.

UserPreferencesResponse

This is the response that is returned when you maintain Favourites lists.

Complex Types

FavouritesKey

This element holds the keys used to identify a Favourites list.

Child elements

Name Minimum Maximum Type Description
UserCode 1 1 string The code that identifies a user master.
CmpCode 1 1 string The code identifying the company in which this Favourites list is held.
Location 1 1 int Identifies the area within the product to which this Favourites list applies.
Context 0 1 string The context in which the Favourites list applies.
ElmLevel 0 1 int The element level for settings.

Favourite

The Favourite type has FavouritesKey as its base type.

This element holds information about a Favourites list, such as the number of lines that will be displayed in the list.

Child elements

Name Minimum Maximum Type Description
MaxLines 1 1 int The maximum number of lines to display in the list.

MasterFavourites

This element holds all the information for a Favourites list.

Child elements

Name Minimum Maximum Type Description
Favourite 1 1 Favourite This element holds information about a Favourites list, such as the number of lines that will be displayed in the list.
List 1 1 MasterFavouritesListData This element holds the items in a Favourites list.

MasterFavouritesListData

This element holds the items in a Favourites list.

Child elements

Name Minimum Maximum Type Description
Item 0 unbounded MasterFavouritesListElement This element holds information about an item in a Favourites list.

MasterFavouritesListElement

This element holds information about an item in a Favourites list.

Child elements

Name Minimum Maximum Type Description
LineNumber 1 1 int The line number of the item.
UserCode 1 1 string The code that identifies a user master.
CmpCode 1 1 string The code identifying the company in which this Favourites list is held.
Location 0 1 int Identifies the area within the product to which this Favourites list applies.
Context 0 1 string The context in which the Favourites list applies
Code 1 1 string The code that identifies a master.
Name 1 1 string The descriptive name for the master.
ShortName 1 1 string The abbreviated name for the master.
ElmLevel 1 1 int The level within the account code if this Favourites item is an element master.
ElmLevelOneOfN 1 1 string An internal representation of the element ElmLevel.
TypeOneOfN 1 1 typeComFavouritesType An internal representation of the element Type.
Type 1 1 int Specifies the item type - whether the row details are derived from the most recently used reports (Dynamic), or defined by the user (Static).
IsDynamic 1 1 boolean Specifies whether the Favourites item is dynamic, which means it is derived from the most recently used Favourites items.
LastAccessed 1 1 dateTime The date and time at which this Favourites item was last selected.
CodeTip 1 1 string The tooltip for the icon beside the Code field when the row type is Static.
