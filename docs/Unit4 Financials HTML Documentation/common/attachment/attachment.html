Attachment

This element holds details of attachments.

Attributes

Name Type Use Default Description
added typeHardTime optional The date and time the attachment was added.
codametadata typeNameTextValues optional List of attachment metadata. These are semi-colon delimited name-value pairs.
externalreference typeTextAnyDbQuotes optional The external reference text for a an externally referenced attachment.
filelastmodified typePosixTime optional The date and time that the attachment was last modified.
filename typeText optional The name of the attached file.
id typeDbGuid optional A unique identifier for the attachment.
mimetype typeText optional The MIME type of the attachment.
missing typeBoolean optional Specifies whether data is missing. True if the client requested for data and none was found; Null if no request was made; False if a request was made and data was found.
path typeText optional The original path of the attached file.
product typeText optional The name of the Coda product that owns this attachment.
ref typeText optional The attachment name.
referenceattachmentdisplayurl typeUri optional The URL used to display the reference attachment.
repositoryobjectid typeDbGuid optional The repository object ID.
thumbnail typeTextAnyUnlimited optional Thumbnail data for an image attachment, in PNG format.
timestamp short required The TimeStamp value for the attachment.
user typeTextAnyUnlimited optional The name of the user who added the attachment.

ObjectUsageSummary

The request for the Find Attachment dialog.

Child elements

Name Minimum Maximum Type Description
Id 1 1 typeDbGuid The ID of the Repository Object returned.
Filename 1 1 typeText The name of the attached file.
User 0 1 typeCode The code of the user who added the attachment.
Added 0 1 typeHardTime The time the attachment was added.
AddedCount 1 1 typeLong Specifies the number of times the object is used.
MimeType 0 1 typeText The MIME type of the attachment.
Application 0 1 typeText The name of the application. If multiple applications use this object then the value of this field is null.
ApplicationCount 1 1 typeLong Specifies the number of applications that use this repository object.
SampleAttachment 0 1 Attachment A sample Attachment object that uses this repository object. This is used because the software can only download things by attachment and not directly from the repository object.
