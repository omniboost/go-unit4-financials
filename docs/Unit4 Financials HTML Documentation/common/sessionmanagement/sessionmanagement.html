http://www.coda.com/common/schemas/sessionmanagement

Root Elements

SessionManagementRequest

This service allows you to manage the current sessions on your installation. Use the GenericBrowseRequest service and the SESSION dataview that Coda provides to get a list of current sessions. Then, if required, delete selected sessions with this service. You can delete only the sessions of users who have a capability level that is lower than your own.

SessionManagementResponse

This is the response that is returned when you delete sessions.

Complex Types

KeyList

Holds a list of session keys that identify the sessions you want to delete.

Child elements

Name Minimum Maximum Type Description
Key 1 unbounded string A session key that identifies a session you want to delete. This is the value of the Session key vocabulary item from the SESSION dataview that Coda provides.

ErrorList

Holds a list of reasons why attempted deletions failed.

Child elements

Name Minimum Maximum Type Description
Error 1 unbounded Reason Holds information about why one attempted deletion failed.
