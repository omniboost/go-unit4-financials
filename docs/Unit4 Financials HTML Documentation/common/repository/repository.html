DataChunk

A chunk of the data for the metadata item. The data is stored as chunks, when persisted to the database, rather than whole since the system will want to stream this data, reading in chunks one-by-one. This enables the system to handle the data without ever requiring it all to be in memory at any one time.

Child elements

Name Minimum Maximum Type Description
Id 1 1 typeDbGuid

The ID of the owning repository object.
Sequence 1 1 typeLong

The sequence number used to ensure the chunks are read in the correct order.
Chunk 1 1 typeText

The chunk of data.
