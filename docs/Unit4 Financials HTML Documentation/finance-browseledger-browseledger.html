BrowseLedgerService

Namespace: http://www.coda.com/efinance/schemas/browseledger/browseledger-13.0/webservice
WSDL: finance-browseledger-browseledger.wsdl
Path: services/finance/browseledger/browseledger-13.0

The following methods are available on this endpoint:

  • Get
  • GetSelector
  • UpdateRow

Get

Input Parameters

GetRequest
Name Minimum Maximum Type Description
LedgerKey 1 1 Key A key used to identify the ledger.
Currency 1 1 typeCtBlrCurrView The currency in which the Browse Ledger results are to be displayed.
Grouping 1 1 Grouping This element holds information about how the Browse Ledger results are to be grouped.
Sort 1 1 SortList This element holds information about the columns used to sort the Browse Ledger results.
SelectedColumns 0 1 ColumnNumberList A list of column numbers indicating which ledger columns are to be retrieved. If this list is not given, all columns are retrieved. A column in the selected columns list may not be returned if it is hidden by grouping.
RowLimit 0 1 typeULong The maximum number of rows that can be retrieved.
ElmKey 0 1 elmFullKey Optionally restricts the ledger to this element.
SelectFilter 1 1 VocListData The selector vocabulary used to retrieve the ledger details.
ColumnFilter 1 1 FilterList This element contains selection criteria used in addition to the criteria contained in the SelectFilter element, but the selection is based on columns in the ledger rather than selector vocabulary. Any column filter which can be expressed as a vocabulary-based filter, will be moved to the SelectFilter by the server to make the selection process more efficient.

Return Value

GetResponse
Name Minimum Maximum Type Description
Filter 1 1 Filter This element contains the criteria used to retrieve ledger details form the database.
LedgerOutOfDate 0 1 boolean Indicates whether the selected ledger master has changed since the last Ledger Update. Some ledger master changes, for example, changing a column title, are ignored.
Selection 0 1 Selection This element contains the ledger details retrieved by Browse Ledger. The selection contains a header row, zero or more data rows, and a totals row.
ReferenceDate 0 1 dateTime The date that Browse Ledger was run for.

GetSelector

Input Parameters

GetSelectorRequest
Name Minimum Maximum Type Description
LedgerKey 1 1 Key A key used to identify the ledger.
OrgCmpCode 0 1 typeGeneralCode The code of the original company for the ledger row.
AccountCode 1 1 typeAcCodeWild The account code for the ledger row.
ColumnNumber 1 1 int The column number.

Return Value

GetSelectorResponse
Name Minimum Maximum Type Description
LedgerKey 1 1 Key A key used to identify the ledger.
OrgCmpCode 0 1 typeGeneralCode The code of the original company for the ledger row.
AccountCode 1 1 typeAcCodeWild The full account code for the ledger row.
ColumnNumber 1 1 int The column number.
SelectFilter 0 1 VocListData The selector for this ledger row and column.

UpdateRow

Input Parameters

UpdateRowRequest
Name Minimum Maximum Type Description
LedgerKey 1 1 Key A key used to identify the ledger.
OrgCmpCode 0 1 typeGeneralCode The code of the original company for the ledger row.
AccountCode 1 1 typeAcCodeWild The full account code for the ledger row.
Currency 1 1 typeCtBlrCurrView The currency in which the Browse Ledger results are to be displayed.
SelectedColumns 0 1 ColumnNumberList A list of column numbers indicating which ledger columns are to be retrieved. If this list is not given, all columns are retrieved. A column in the selected columns list may not be returned if it is hidden by grouping.

Return Value

UpdateRowResponse
Name Minimum Maximum Type Description
LedgerKey 1 1 Key A key used to identify the ledger.
OrgCmpCode 0 1 typeGeneralCode The code of the original company for the ledger row.
AccountCode 1 1 typeAcCodeWild The full account code for the ledger row.
Currency 1 1 typeCtBlrCurrView The currency in which the Browse Ledger results are displayed.
SelectedColumns 0 1 ColumnNumberList This element contains a list of column numbers indicating which ledger columns have been retrieved.
Selection 0 1 Selection This element contains the revised ledger data resulting from the update. The selection contains a header row and one data row (no SelectFilter or ColumnFilter is applied to this data).
